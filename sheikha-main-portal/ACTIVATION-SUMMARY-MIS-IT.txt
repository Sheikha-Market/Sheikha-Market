🚀 كيفية الاستخدام الفوري
═══════════════════════════════════════════════════════════════════════════════════════════

1️⃣ تشغيل الخادم:
   $ npm start

   أو مع إعادة التحميل التلقائي:
   $ npm run dev

2️⃣ اختبار MIS Endpoints:
   $ curl http://localhost:8080/api/mis/overview

   أو استخدم السكريبت المتقدم:
   $ bash test-curl-sheikha.sh

3️⃣ اختبار IT Endpoints:
   $ curl http://localhost:8080/api/it/overview

4️⃣ تشغيل مجموعة الاختبارات الكاملة:
   $ chmod +x test-sheikha-systems.sh
   $ ./test-sheikha-systems.sh
