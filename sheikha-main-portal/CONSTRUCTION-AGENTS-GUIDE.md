## 🏗️ نظام منظومة وكلاء البناء والتطوير الشامل

### Comprehensive Construction & Development Project Management Agents System

---

## 📋 مرحباً بك في منظومة الوكلاء المتكاملة

**منظومة شيخة لإدارة المشاريع والبناء والتطوير** هي نظام ذكي متكامل لإدارة المشاريع الإنشائية والتطويرية بكل جوانبها، مع الالتزام الكامل بالمبادئ الإسلامية والمعايير الدولية.

### 🎯 الأهداف الرئيسية

✅ **عمارة الأرض** - إصلاح وتطوير العمران باحترافية عالية
✅ **الجودة والابتكار** - أفضل المعايير العالمية مع الشريعة الإسلامية
✅ **السلامة والأمان** - حفظ الأرواح والممتلكات
✅ **الكفاءة** - عدم إهدار الموارد والتطوير المستمر
✅ **الاستدامة** - العمل بما ينفع الأجيال القادمة

## 👥 هيكل الوكلاء (34 وكيل متخصص)

### 1️⃣ المستوى التنفيذي (1)

**👔 مدير المشروع العام** `agent-project-general-manager`

- الإشراف الشامل على المشروع
- وضع الاستراتيجية والخطط الإجمالية
- إدارة الميزانية والموارد
- التنسيق بين جميع الأقسام

### 2️⃣ رؤساء الأقسام (3)

#### 🏢 مدير قسم البناء والتشييد `agent-construction-manager`

- الإشراف على جميع أعمال البناء
- إدارة فريق العمال والمقاولين
- مراقبة مطابقة العمل للمخططات
- إدارة السلامة على الموقع

#### 🔧 مدير الهندسة والتصميم `agent-engineering-manager`

- الإشراف على جميع الأعمال الهندسية
- إدارة فريق المهندسين
- التحقق من المخططات والتصاميم
- توجيه المهندسين المتخصصين

#### ⚙️ مدير العمليات والتنسيق `agent-operations-manager`

- تنسيق جميع العمليات اليومية
- إدارة الموارد والمعدات
- تحسين الكفاءة والإنتاجية
- إدارة السلاسل اللوجستية

### 3️⃣ المشرفون المتخصصون (6)

- **مشرف الموقع الميداني** `agent-site-supervisor` - الإشراف المباشر اليومي
- **مشرف الهندسة الإنشائية** `agent-structural-engineer-supervisor` - الإشراف على الأعمال الإنشائية
- **مشرف نظام الكهرباء** `agent-electrical-supervisor` - الإشراف على الأنظمة الكهربائية
- **مشرف الأنظمة الميكانيكية** `agent-mechanical-supervisor` - الإشراف على الأنظمة الميكانيكية
- **مشرف السلامة والصحة المهنية** `agent-safety-officer-general` - إدارة السلامة الشاملة
- **مفتش الجودة والمواصفات** `agent-quality-inspector-construction` - التفتيش على الجودة

### 4️⃣ المهندسون المتخصصون (8)

- **مهندس الإنشاء والبناء** - تصميم وحساب الأساسات والأعمدة
- **مهندس المدني والعام** - البنية التحتية والطرق والجسور
- **مهندس الكهرباء والقوى** - تصميم الأنظمة الكهربائية
- **مهندس الميكانيكا والأنظمة** - نظم التكييف والتدفئة والطاقة
- **مهندس HVAC والتهوية** - نظم التكييف والتهوية المتقدمة
- **مهندس السباكة والمياه** - نظم إمدادات المياه والصرف الصحي
- **مهندس العمارة والتصميم** - التصميم المعماري والديكور
- **مهندس الأنظمة والتقنية** - الأنظمة الذكية والتحكم

### 5️⃣ الفنيون والعمال المتخصصون (8)

- **مساعد المهندس في الموقع** - مساعدة العمل الهندسي
- **رئيس النجارين** - الإشراف على أعمال النجارة والقوالب
- **رئيس الكهربائيين** - الإشراف على الأعمال الكهربائية
- **رئيس السباكين** - الإشراف على أعمال السباكة
- **متخصص الخرسانة والصب** - تحضير وصب الخرسانة
- **رئيس عمال الحديد والفولاذ** - تركيب الحديد واللحام
- **متخصص التشطيبات والديكور** - الدهان والديكور والتشطيبات
- **فني السلامة والحماية** - توفير وإدارة معدات الحماية

### 6️⃣ خدمات الإدارة والدعم (5)

- **منسق المقاولين والعقود** - إدارة العقود والمقاولين
- **مدير المشتريات والتوريد** - تخطيط وتوريد المواد والمعدات
- **مدير الموارد البشرية** - التوظيف والرواتب والتدريب
- **مراقب الحسابات والمالية** - إدارة الميزانية والحسابات
- **متخصص التوثيق والأرشفة** - توثيق وحفظ المستندات

### 7️⃣ فريق البحث والتطوير والابتكار (3)

- **مدير البحث والتطوير** - قيادة فريق التطوير والابتكار
- **مهندس الابتكار والحلول** - تطوير حلول جديدة ومبتكرة
- **متخصص الاستدامة والبيئة** - ضمان الممارسات المستدامة

## 🔌 API Endpoints

### 1. الحصول على قائمة جميع الوكلاء

```
GET /api/construction-agents/list
```

**الرد:**

```json
{
  "success": true,
  "data": {
    "summary": {
      "totalAgents": 34,
      "executives": 1,
      "departmentHeads": 3,
      "supervisors": 6,
      "engineers": 8,
      "technicians": 8,
      "administrative": 5,
      "rdDevelopment": 3
    },
    "agents": [...]
  }
}
```

### 2. الحصول على معلومات وكيل محدد

```
GET /api/construction-agents/:agentId
```

**مثال:**

```
GET /api/construction-agents/agent-project-general-manager
```

### 3. تفعيل وكيل واحد

```
POST /api/construction-agents/activate
```

**البيانات المطلوبة:**

```json
{
    "agentId": "agent-project-general-manager",
    "config": {
        "projectName": "مشروع البرج A",
        "location": "الرياض",
        "budget": 150000000
    }
}
```

### 4. تفعيل جميع الوكلاء (التفعيل الشامل)

```
POST /api/construction-agents/activate-all
```

**البيانات المطلوبة:**

```json
{
    "projectName": "مشروع البرج التجاري A",
    "config": {}
}
```

**الرد:**

```json
{
  "success": true,
  "message": "تم تفعيل 34/34 وكيل",
  "data": {
    "summary": {
      "totalActivated": 34,
      "totalFailed": 0,
      "projectName": "مشروع البرج التجاري A"
    },
    "activatedAgents": [...]
  }
}
```

### 5. تفعيل وكلاء حسب الدور

```
POST /api/construction-agents/activate-by-role
```

**البيانات المطلوبة:**

```json
{
    "role": "EXECUTIVE_LEADER",
    "config": {}
}
```

**الأدوار المتاحة:**

- `EXECUTIVE_LEADER` - المديرون التنفيذيون
- `DEPARTMENT_HEAD` - رؤساء الأقسام
- `SUPERVISOR` - المشرفون
- `SPECIALIST` - المهندسون
- `TECHNICIAN` - الفنيون والعمال
- `ADMINISTRATIVE` - الإدارة والدعم
- `STRATEGIC_DEVELOPMENT` - البحث والتطوير

### 6. الحصول على ملخص إدارة المشروع

```
GET /api/construction-agents/project-summary
```

## 💡 أمثلة عملية

### مثال 1: تفعيل مدير المشروع

```bash
curl -X POST http://localhost:8080/api/construction-agents/activate \
  -H "Content-Type: application/json" \
  -d '{
    "agentId": "agent-project-general-manager",
    "config": {
      "projectName": "مشروع العمارة الذكية",
      "location": "جدة",
      "budget": 200000000,
      "duration": "24 months"
    }
  }'
```

### مثال 2: تفعيل فريق الهندسة كاملاً

```bash
curl -X POST http://localhost:8080/api/construction-agents/activate-by-role \
  -H "Content-Type: application/json" \
  -d '{
    "role": "DEPARTMENT_HEAD",
    "config": {}
  }'
```

### مثال 3: تفعيل جميع الوكلاء للمشروع

```bash
curl -X POST http://localhost:8080/api/construction-agents/activate-all \
  -H "Content-Type: application/json" \
  -d '{
    "projectName": "مشروع البناء المتكامل 2026",
    "config": {
      "city": "الرياض",
      "startDate": "2026-03-15"
    }
  }'
```
