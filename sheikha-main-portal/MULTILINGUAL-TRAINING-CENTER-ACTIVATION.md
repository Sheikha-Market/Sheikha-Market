### 4️⃣ أفضل تقنيات التدريب العالمية

التقنيات المُدمَجة:

1. **LoRA** (Low-Rank Adaptation)
    - كفاءة عالية في التدريب
    - استهلاك منخفض للذاكرة
    - ملائم للتدريب المستمر

2. **QLoRA** (Quantized LoRA)
    - استهلاك ذاكرة أقل بـ 4x
    - مثالي للأجهزة المحدودة

3. **PEFT** (Parameter-Efficient Fine-Tuning)
    - تدريب فعّال
    - توازن مثالي بين الأداء والموارد

4. **DPO** (Direct Preference Optimization)
    - تحسين مباشر للتفضيلات
    - دقة عالية

5. **RLHF** (Reinforcement Learning from Human Feedback)
    - تعلم تعزيزي
    - تحسين مستمر

6. **ADAPTER_TUNING** - ضبط دقيق بمحولات
7. **PREFIX_TUNING** - ضبط البادئات
8. **PROMPT_TUNING** - ضبط المحفزات

## 🔌 واجهات برمجة التطبيقات (APIs)

### مركز التدريب (Training Center)

```bash
# الإحصائيات
GET /api/training-center/stats

# قائمة اللغات
GET /api/training-center/languages

# تدريب شامل بكل اللغات
POST /api/training-center/train-all-languages
{
  "agentId": "agent-deen-preservation",
  "batchSize": 10,
  "epochs": 5,
  "technique": "LORA"
}

# تدريب بتقنية محددة
POST /api/training-center/train-with-technique
{
  "agentId": "agent-deen-preservation",
  "technique": "QLORA",
  "languages": ["ARABIC", "ENGLISH"],
  "batchSize": 8,
  "epochs": 3
}

# جدولة التدريب
POST /api/training-center/schedule-training
{
  "agentId": "agent-deen-preservation",
  "scheduleType": "CONTINUOUS",
  "config": {
    "technique": "LORA",
    "batchSize": 16
  }
}

# إنشاء وكيل إداري
POST /api/training-center/create-management-agent
{
  "type": "SCHEDULER"
}
```

### الوكلاء الإسلاميون (Islamic Agents)

```bash
# تفعيل جميع الوكلاء
POST /api/islamic-agents/activate-all

# قائمة الوكلاء
GET /api/islamic-agents/list

# تنفيذ مهمة
POST /api/islamic-agents/execute-task
{
  "agentId": "agent-deen-preservation",
  "task": {
    "type": "fatwa",
    "question": "ما حكم..."
  }
}
```

### تحسين LLM (LLM Optimization)

```bash
# هندسة المحفزات
POST /api/llm-optimization/engineer-prompt
{
  "text": "اشرح مفهوم...",
  "technique": "CHAIN_OF_THOUGHT"
}

# تحسين اللغة
POST /api/llm-optimization/enhance-language
{
  "text": "النص المراد تحسينه",
  "targetLanguage": "ARABIC"
}

# ضبط دقيق
POST /api/llm-optimization/finetune-model
{
  "dataset": [...],
  "epochs": 5
}
```
