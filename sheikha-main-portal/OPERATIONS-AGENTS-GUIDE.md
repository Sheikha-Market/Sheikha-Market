## 📋 نظرة عامة

منظومة **SHEIKHA** للعمليات والإدارة تضم **47 وكيل متخصص** موزعين على **10 أقسام رئيسية**، مبنية بالكامل على مبادئ **الكتاب والسنة** ومعايير الجودة الدولية.

### 🎯 الأهداف الاستراتيجية

1. **الامتثال الشرعي**: جميع العمليات متوافقة مع الشريعة الإسلامية
2. **الكفاءة التشغيلية**: تحسين جميع العمليات والموارد
3. **الجودة العالية**: تطبيق معايير ISO وأفضل الممارسات
4. **الشفافية والنزاهة**: العدل والأمانة في جميع المعاملات
5. **التطوير المستمر**: التدريب والتطوير لجميع الوكلاء

## 🏗️ الهيكل التنظيمي

### 1️⃣ قسم سلاسل الإمداد والموارد (SUPPLY_CHAIN_MANAGEMENT)

**عدد الوكلاء: 4**

#### الوكلاء:

- **مدير سلاسل الإمداد** (`supply_chain_director`)
    - قيادة استراتيجية سلاسل الإمداد
    - KPIs: وقت دورة التوصيل، مستوى المخزون، رضا الموردين

- **مدير اللوجستيات والنقل** (`logistics_manager`)
    - تخطيط وتنسيق عمليات النقل
    - KPIs: معدل التسليم في الوقت، تكلفة النقل، معدل تلف البضائع

- **مدير المخازن والتخزين** (`warehouse_manager`)
    - إدارة عمليات المخازن وأنظمة WMS
    - KPIs: دقة المخزون، معدل الدوران، استغلال المساحة

- **مراقب المخزون** (`inventory_controller`)
    - مراقبة مستويات المخزون والجرد
    - KPIs: دقة الجرد، نسبة المخزون الراكد

**المبادئ الإسلامية:**

- الأمانة في التعاملات التجارية
- الصدق والوفاء بالعهود
- العدل في الأسعار وعدم الاحتكار

### 2️⃣ قسم إدارة الموارد والمواد (RESOURCE_MANAGEMENT)

**عدد الوكلاء: 3**

- **مدير المواد والمشتريات** (`materials_manager`)
    - تخطيط احتياجات المواد وإدارة الموردين
    - KPIs: نسبة توفر المواد، جودة المواد، توفير التكاليف

- **أخصائي المشتريات** (`procurement_specialist`)
    - البحث عن الموردين والتفاض
    - KPIs: وقت معالجة الطلبات، نسبة التوفير، دقة الطلبات

**المبادئ الإسلامية:**

- الصدق في المواصفات
- العدل في التعامل
- تجنب الغش والتدليس

### 4️⃣ قسم المحاسبة والمالية (FINANCE_ACCOUNTING)

**عدد الوكلاء: 4**

#### الوكلاء:

- **المدير المالي التنفيذي (CFO)** (`cfo`)
    - قيادة الاستراتيجية المالية
    - KPIs: الربحية (ROI/ROE), السيولة، دقة التنبؤات

- **المراقب المالي** (`financial_controller`)
    - مراقبة العمليات المالية اليومية
    - KPIs: دقة التقارير، وقت إغلاق الحسابات، نسبة الأخطاء

- **محاسب** (`accountant`)
    - إدخال القيود وإعداد التقارير
    - KPIs: دقة الإدخالات، وقت المعاملات، دقة المطابقة

- **محاسب الزكاة والتمويل الإسلامي** (`zakat_calculator`)
    - حساب الزكاة وإدارة التمويل الإسلامي
    - KPIs: دقة حساب الزكاة، نسبة المعاملات الشرعية

**المبادئ الإسلامية:**

- الصدق في التقارير المالية
- تجنب الربا والغرر
- حساب الزكاة وإخراجها
- الشفافية المالية
- تطبيق المعايير الشرعية (AAOIFI)

**المبادئ الإسلامية:**

- الأمانة في الرقابة

**عدد الوكلاء: 5**

#### الوكلاء:

- **مدير الموارد البشرية للوكلاء** (`hr_director`)

    - KPIs: معدل رضا الوكلاء، معدل الاحتفاظ، الإنتاجية، فعالية التدريب

- **أخصائي التوظيف** (`recruitment_specialist`)
    - البحث عن المواهب وجذبها
    - KPIs: وقت التوظيف، جودة التوظيف، تكلفة التوظيف

- **مدير التدريب والتطوير** (`training_manager`)

    - رقمنة التدريب بالكتاب والسنة
    - KPIs: معدل الإنجاز، تحسن الأداء، رضا المتدربين، عائد الاستثمار

- **مدير المواهب** (`talent_manager`)
    - تحديد المواهب وتطوير خطط التعاقب
    - KPIs: معدل الاحتفاظ بالمواهب، نسبة الترقيات الداخلية

- **محلل الأداء** (`performance_analyst`)
    - قياس وتحليل أداء الوكلاء
    - KPIs: دقة القياس، معدل تحسن الأداء، رضا الوكلاء

**المبادئ الإسلامية:**

- العدل في التوظيف
- الرحمة والرفق بالعاملين
- إعطاء الأجر قبل أن يجف العرق
- طلب العلم فريضة
- الإتقان في العمل
- التزكية والتطوير المستمر

## 📚 الاستخدام

### تحميل المحرك:

```javascript
const OpsEngine = require('./lib/sheikha-operations-management-agents-engine.js');
```

### الحصول على جميع الوكلاء:

```javascript
const allAgents = OpsEngine.getAllOperationsAgents();
console.log(`إجمالي الوكلاء: ${allAgents.length}`);
```

### الحصول على وكلاء قسم معين:

```javascript
const hrAgents = OpsEngine.getAgentsByRole('AGENT_HR');
const healthAgents = OpsEngine.getAgentsByRole('HEALTHCARE');
```

### البحث عن وكيل:

```javascript
const agent = OpsEngine.getAgentById('agent-cfo');
const searchResults = OpsEngine.searchAgents('محاسب');
```

### تفعيل وكيل:

```javascript
const activeAgent = OpsEngine.activateAgent('agent-training-manager', {
    language: 'ar',
    mode: 'production'
});
```

### الحصول على إحصائيات:

```javascript
const stats = OpsEngine.getAgentsStatistics();
console.log(stats);
```
