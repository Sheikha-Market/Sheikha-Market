## 📌 **Tasks الجاهزة في VS Code**

يمكنك تشغيلها من **Tasks: Run Task** (`Ctrl+Shift+P`):

1. **📦 Install Dependencies** - تثبيت npm packages
2. **🚀 Start Development Server** - تشغيل خادم التطوير
3. **🔧 Run Scripts Check** - فحص الجاهزية
4. **🎯 Run VSCode Doctor** - تشخيص البيئة ✅ (جاهز)
5. **🟢 Start Dev (No Cursor)** - تشغيل بدون Cursor
6. **🛡️ Full Secure Activate** - تفعيل آمن كامل
