# هوية الأفضل — تناسب المنظومة والسوق والتخصص

## النطاق والهدف

- **هوية الأفضل:** هوية واضحة **تناسب المنظومة** (شيخه المتكاملة)، **السوق** (معادن وسكراب وإعادة تدوير)، و**التخصص** (حديد، نحاس، ألمنيوم، مصاهر، سلاسل توريد، لوجستيات) — بحيث يظهر الموقع والعلامة كـ **الأفضل** في هذا المجال دون التباس.
- **المنظومة:** سوق شيخه + لوحة تحكم + تكامل ERP/SCM + ذكاء وشات + لوائح وهوية إسلامية — **منظومة واحدة متكاملة**.
- **السوق:** سوق **المعادن والسكراب** — شراء، بيع، عرض، توريد، إعادة تدوير.
- **التخصص:** تخصص واحد واضح — **معادن · سكراب · إعادة تدوير · سلاسل توريد** (حديد، نحاس، ألمنيوم، مصاهر، لوجستيات).

## 4. قواعد التطبيق

- **كل صفحة:** الشريط العلوي أو الهيدر يعرّف "سوق شيخه للمعادن والسكراب" و"تخصص: معادن · سكراب · إعادة تدوير" حيث يلزم.
- **الصفحة الرئيسية:** البطل والفوتر يعكسان المنظومة والسوق والتخصص والهوية الإسلامية دون اختصار يسبب التباساً.
- **اللوائح والهوية:** وثائق هوية-شيخه-المتكاملة والتصميم — مرجع الألوان والخط والقواعد الحلال.
