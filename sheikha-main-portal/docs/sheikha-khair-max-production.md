# شيخة: الإنتاج الأقصى الآمن

هذا المسار يحقق أقصى إنتاج تقني ممكن داخل الأصول المملوكة أو المفوضة فقط.

## المبادئ
- الكتاب والسنة
- لا ضرر ولا ضرار
- التوحيد والأمانة والعدل
- التحسين المستمر
- تشغيل ما هو مصرح به فقط

## الأنماط
- نمط زمني: مراجعة شهرية تلقائية.
- نمط معياري: ترقية عند تقنية جديدة أو إضافة جديدة أو هدف ابتكار جديد.
- نمط قيمي: التنافس في الخير والمنفعة والابتكار الآمن.

## التشغيل
- تخطيط فقط:
  - `npm run ops:khair:max:plan`
- تفعيل فعلي:
  - `npm run ops:khair:max:apply`
- فحص الحالة:
  - `npm run ops:khair:max:status`

## ما الذي يفعله
- يفعّل نوى الإنتاج والتوليد والحوسبة المتقدمة.
- يبدأ حراس الخلفية عبر PM2 عندما تكون متاحة.
- يفعّل حماية الهوية والأسرار ومراقبة الخلفية وسلامة الملكية.
- يولد تقريرًا موحدًا في `data/sheikha-khair-max-production-report.json`.

## حدود صريحة
- لا يتحكم في خوادم أو سحابات غير مملوكة أو غير مفوضة.
- لا ينفذ أوامر تخريبية.
- لا يتجاوز الصلاحيات القانونية أو التقنية.
