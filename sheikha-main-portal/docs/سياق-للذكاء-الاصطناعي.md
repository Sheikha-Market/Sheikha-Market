## 🎯 وصف المشروع

**منظومة شيخة** هي منصة سعودية رائدة لتجارة المعادن والسكراب، تقدم:

1. **سوق إلكتروني** لبيع وشراء المعادن والسكراب
2. **مؤشر أسعار عادل (SMI)** قائم على مبادئ إسلامية
3. **نظام ERP/SCM** لإدارة سلسلة التوريد
4. **منتدى علمي** للباحثين والمختصين
5. **مساعد ذكي (شيخة AI)** للدعم والإرشاد
6. **جمعية شيخة** لتنظيم القطاع وتشريع الأسعار

## ⚖️ نظام التسعير العادل

```javascript
// الثوابت (لا تتغير)
FPM: 5-15%   // هامش الربح العادل
PPF: 1.05    // حماية المنتج (الحد الأدنى)
CPF: 1.25    // حماية المستهلك (الحد الأقصى)

// المتغيرات (تتغير لحظياً)
SDV: العرض والطلب
GPV: الأسعار العالمية
SV: الموسمية

// المعادلة
السعر العادل = تكلفة × (1 + FPM) × SDV × GPV × SV
```

## 🎯 أنواع الطلبات الشائعة

عند التطوير، قد أطلب:

1. **إضافة API جديد** → عدّل `server.js`
2. **تعديل صفحة** → عدّل ملف في `public/`
3. **تحديث الذكاء** → عدّل `lib/sheikha-ai.js`
4. **إضافة توثيق** → أنشئ ملف في `docs/`
5. **تصحيح خطأ** → حدد الملف والسطر
