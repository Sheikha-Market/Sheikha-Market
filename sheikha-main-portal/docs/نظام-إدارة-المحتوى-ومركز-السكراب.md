# نظام ومنظومة إدارة المحتوى — مركز أسواق السكراب وأول منتج: بيع السكراب

## 1) الهدف

- **تفعيل نظام ومنظومة إدارة المحتوى** — محتوى قوي واحترافي، الأفضل بالموقع.
- **سوق شيخه مركز وأساس أسواق السكراب في مكان واحد** — يجذب كل الأسواق في السعودية والعالم، لكل سوق عالمي.
- **البدء بأول منتج: بيع السكراب** — تداول وبيع السكراب العالمي، جذب كل الأسواق، طريقة لعرض المنتجات والأسواق والكميات والأنواع، **تسجيل المنتجات آلياً** وتحقيق أرباح.

---

## 2) نظام ومنظومة إدارة المحتوى

| العنصر | الوصف |
|--------|--------|
| **نظام إدارة المحتوى** | محتوى قوي واحترافي — عناوين، نصوص، دعوات واضحة (أول منتج: بيع السكراب، مركز أسواق السكراب). |
| **منظومة إدارة المحتوى** | تعمل بالخلفية — توحيد الصياغة، التحديث، والربط مع واجهة المحتوى (مثلاً من API `/api/content-config`). |

المحتوى المعروض في الصفحة الرئيسية (Hero، قسم الانطلاق، قسم السوق) يعكس: مركز أسواق السكراب، أول منتج بيع السكراب، المنتجات والأسواق والكميات والأنواع، وتسجيل المنتجات آلياً وتحقيق أرباح.

---

## 3) سوق شيخه مركز أسواق السكراب

- **مكان واحد** — يجمع عروض البضائع من متاجر وتجار في السعودية والعالم.
- **يجذب كل الأسواق** — السعودية، الخليج، العالم — لكل سوق عالمي.
- **تداول وبيع السكراب العالمي** — حديد، نحاس، ألومنيوم، رصاص، زنك، ستانلس، سكراب مختلط.
- **عرض المنتجات والأسواق والكميات والأنواع** — في قسم «المنتجات والأسواق والكميات والأنواع» مع فلتر حسب النوع (حديد، نحاس، ألومنيوم، إلخ).

---

## 4) أول منتج: بيع السكراب

- **البدء الآن** — أول منتج في الموقع هو **بيع السكراب**.
- **طريقة العرض** — عروض بضائع (اسم المنتج، المتجر/التاجر، المنطقة، الكمية، الوحدة، السعر، النوع).
- **تسجيل المنتجات آلياً** — عند إضافة عرض بضائع عبر `POST /api/listings`، يُصنَّف المنتج آلياً حسب اسم البضاعة (كلمات مفتاحية: حديد، نحاس، ألومنيوم، رصاص، زنك، ستانلس، سكراب، إلخ) ويُحفظ التصنيف مع العرض.
- **تحقيق أرباح** — عبر عمولات المنصة، اشتراكات، استشارات، نقل، وتداول ذاتي (انظر نموذج-عمل-مبتكر-وتحقيق-أرباح.md).

---

## 5) واجهات API ذات الصلة

| المسار | الوصف |
|--------|--------|
| **GET /api/content-config** | إعدادات المحتوى (Hero، أول منتج، مركز الأسواق). |
| **GET /api/market-listings?type=iron** | عروض البضائع مع التصنيف الآلي؛ اختياري: فلتر حسب النوع (iron, copper, aluminum, …). |
| **GET /api/scrap-categories** | قائمة أنواع السكراب المستخدمة في التصنيف الآلي. |
| **POST /api/listings** | إضافة عرض بضائع — يتم تسجيل المنتج وتصنيفه آلياً. |

---

## 6) حالة الأنظمة

في `GET /api/systems-status`:

- **contentManagement:** نظام إدارة المحتوى — محتوى قوي واحترافي.
- **contentEcosystem:** منظومة إدارة المحتوى — بالخلفية.
- **scrapHub:** مركز أسواق السكراب — مكان واحد يجذب كل الأسواق السعودية والعالم.
