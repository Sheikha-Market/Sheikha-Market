## 1. ما هو CUDA؟

**NVIDIA CUDA** (Compute Unified Device Architecture — بنية الحوسبة الموحدة للأجهزة) منصة حوسبة متوازية وواجهة برمجة تطبيقات (API) تتيح للمطورين استخدام القوة الحسابية الهائلة لوحدات معالجة الرسومات (GPU) لأغراض عامة تتجاوز مجرد عرض الرسوميات.

## 4. مسارات التكامل

### 4.1 خدمة Docker مع CUDA

```dockerfile
FROM nvidia/cuda:12.0-runtime-ubuntu22.04
# PyTorch/TensorFlow على GPU
```

**الحالة:** جاهزية — بنية Node.js قابلة للربط

### 4.2 جسر Node.js → CUDA

- استدعاء خدمات Python/CUDA من `server.js` عبر `child_process` أو API داخلي
- مسار: `lib/` أو خدمة منفصلة على منفذ آخر

### 4.3 NVIDIA NIM API

- استدلال سحابي — لا يحتاج GPU محلي
- **الحالة:** قيد التفعيل — `NVIDIA_API_KEY`
