# بسم الله الرحمن الرحيم

# دليل التثبيت والتوزيع — شيخة

## 1) التثبيت الرسمي (Terminal)
```bash
sudo bash scripts/install.sh
```

## 2) إزالة التثبيت
```bash
sudo bash scripts/uninstall.sh
```

## 3) بناء حزمة Debian
```bash
bash scripts/build-deb.sh
sudo dpkg -i dist/sheikha-main-portal_1.0.0_amd64.deb
```

## 4) التحقق الأمني قبل الإصدار
```bash
bash scripts/validate-release.sh
bash scripts/release-checksums.sh
```

## 5) فحص الخدمة
```bash
systemctl status sheikha --no-pager
curl -fsS http://127.0.0.1:8080/api/health
```

## 6) التوجيهات التشغيلية
- لا تشغّل الخدمة بأسرار افتراضية.
- فعّل `SECURITY_STRICT_SECRETS=true` في الإنتاج.
- استخدم HTTPS مع `HSTS_ENABLED=true`.
- فعّل فحص الملفات الضارة عند وجود رفع ملفات.
