# بسم الله الرحمن الرحيم

# دليل التثبيت والتشغيل والتحديث — شيخة

## 1) التثبيت السريع
```bash
curl -fsSL https://www.sheikha.top/scripts/install.sh -o install.sh
sudo bash install.sh
```

## 2) تشغيل الخدمة
```bash
sudo systemctl status sheikha
sudo systemctl restart sheikha
sudo journalctl -u sheikha -n 100 --no-pager
```

## 3) التهيئة
- عدّل `/opt/sheikha/sheikha-main-portal/.env`
- عيّن:
  - `JWT_SECRET`
  - `DATA_ENCRYPTION_KEY_HEX`
  - مفاتيح التكامل المطلوبة.

## 4) التحقق
```bash
curl -fsS http://127.0.0.1:8080/api/health
curl -fsS http://127.0.0.1:8080/api/integrations/development/status
```

## 5) التحديث
```bash
cd /opt/sheikha/sheikha-main-portal
npm ci --omit=dev
npm run server:deploy:safe
```

## 6) الاسترجاع السريع (Rollback)
```bash
cd /opt/sheikha/sheikha-main-portal
npm run server:rollback:last-good
sudo systemctl restart sheikha
```

## 7) الإزالة
```bash
sudo bash scripts/uninstall.sh
# إزالة كاملة مع البيانات:
sudo SHEIKHA_PURGE_DATA=true bash scripts/uninstall.sh
```
