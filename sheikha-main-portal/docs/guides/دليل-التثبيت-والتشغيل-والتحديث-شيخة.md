# بسم الله الرحمن الرحيم

# دليل التثبيت والتشغيل والتحديث — شيخة

## 1) التثبيت السريع
```bash
curl -fsSL https://www.sheikha.top/scripts/install.sh -o install.sh
sudo bash install.sh
```

## 2) تشغيل الخدمة
```bash
sudo systemctl status sheikha
sudo systemctl restart sheikha
sudo journalctl -u sheikha -n 100 --no-pager
```

## 3) التهيئة
- عدّل `/opt/sheikha/sheikha-main-portal/.env`
- عيّن:
  - `JWT_SECRET`
  - `DATA_ENCRYPTION_KEY_HEX`
  - مفاتيح التكامل المطلوبة.

## 4) التحقق
```bash
curl -fsS http://127.0.0.1:8080/api/health
curl -fsS http://127.0.0.1:8080/api/integrations/development/status
```

## 5) التحديث
### 5.1 تأمين فوري
```bash
# 1) ألغِ أي PAT قديم ظهر في الطرفية (Bad credentials) من GitHub Settings > Developer settings
# 2) لا تنشئ Token جديد إلا عند الحاجة لاستدعاءات API فقط
# 3) اعتمد SSH فقط لعمليات Git
```

### 5.2 تثبيت إعداد Git على السيرفر
```bash
git -C /opt/sheikha/current remote -v
git -C /opt/sheikha/current rev-parse --abbrev-ref HEAD
```

### 5.3 نشر آمن (موصى به)
```bash
cd /opt/sheikha/sheikha-main-portal
npm ci --omit=dev
npm run server:deploy:safe
```

### 5.4 نشر آمن يدوي (Pull)
```bash
cd /opt/sheikha/current
git rev-parse HEAD > /opt/sheikha/last_good_commit.txt
git -C /opt/sheikha/current fetch origin
git -C /opt/sheikha/current checkout main
git -C /opt/sheikha/current pull --ff-only origin main
```

### 5.5 Restart للخدمة
```bash
# PM2
pm2 restart sheikha-main-portal && pm2 save

# أو systemd
systemctl restart sheikha-main-portal
systemctl status sheikha-main-portal --no-pager -l
```

### 5.6 Healthcheck بعد النشر
```bash
# فحص داخلي
curl -fsS http://127.0.0.1:8080/api/health

# فحص خارجي
curl -fsS https://your-domain.example/api/health
```

## 6) الاسترجاع السريع (Rollback)
```bash
cd /opt/sheikha/sheikha-main-portal
npm run server:rollback:last-good
sudo systemctl restart sheikha
```

## 7) الإزالة
```bash
sudo bash scripts/uninstall.sh
# إزالة كاملة مع البيانات:
sudo SHEIKHA_PURGE_DATA=true bash scripts/uninstall.sh
```
