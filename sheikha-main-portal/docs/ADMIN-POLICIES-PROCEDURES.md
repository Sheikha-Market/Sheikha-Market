## 🏛️ الفصل الأول: الإطار العام للحوكمة

### 1.1 الرؤية والرسالة

**الرؤية:**
أن نكون المنصة الرائدة عالمياً في تجارة المعادن الرقمية، ملتزمين بأعلى معايير الجودة والشفافية وأحكام الشريعة الإسلامية.

**الرسالة:**
تمكين التجار والمصانع والمستهلكين من إجراء معاملات تجارية آمنة وشفافة في قطاع المعادن، مع توفير حلول رقمية متكاملة.

**القيم الأساسية:**
| القيمة | التعريف |
|--------|---------|
| الأمانة | الصدق في جميع التعاملات |
| الشفافية | الوضوح الكامل في الأسعار والشروط |
| الجودة | تقديم أعلى مستوى من الخدمة |
| الالتزام الشرعي | التوافق مع أحكام الشريعة الإسلامية |
| الابتكار | التطوير المستمر بأحدث التقنيات |

## 📜 الفصل الثاني: السياسات العامة

### 2.1 سياسة التسجيل والقبول

#### 2.1.1 شروط قبول التجار
```yaml
المتطلبات الإلزامية:
  - سجل تجاري ساري المفعول
  - شهادة الزكاة والدخل
  - رخصة تجارة المعادن (إن وجدت)
  - هوية المالك أو المفوض
  - عنوان فعلي للنشاط
  - حساب بنكي تجاري

معايير التقييم:
  - سمعة تجارية جيدة
  - عدم وجود قضايا غش أو تزوير
  - القدرة المالية الكافية
  - الالتزام بالتعاملات الشرعية
```

#### 2.1.2 مستويات التحقق
| المستوى | المتطلبات | الصلاحيات |
|---------|----------|-----------|
| أساسي | هوية + جوال | تصفح فقط |
| موثق | + سجل تجاري | شراء حتى 50,000 ر.س |
| معتمد | + زيارة ميدانية | شراء/بيع بدون حدود |
| VIP | + تاريخ تعاملات | مزايا حصرية |

#### 2.1.3 إجراء التسجيل المؤتمت
```
الخطوة 1: تقديم الطلب إلكترونياً
    ↓ [تلقائي: 30 ثانية]
الخطوة 2: التحقق من السجل التجاري (API)
    ↓ [تلقائي: دقيقة]
الخطوة 3: التحقق من أبشر/الهوية
    ↓ [تلقائي: دقيقة]
الخطوة 4: الموافقة المبدئية
    ↓ [تلقائي]
الخطوة 5: تفعيل الحساب وإرسال بيانات الدخول
    ↓ [تلقائي: فوري]
✅ الحساب جاهز للاستخدام
```

### 2.2 سياسة التسعير والعمولات

#### 2.2.1 جدول العمولات
| الباقة | الرسوم الشهرية | العمولة | المزايا |
|--------|---------------|---------|---------|
| مجاني | 0 ر.س | 5% | أساسية |
| أساسي | 500 ر.س | 3% | + دعم أولوية |
| احترافي | 1,500 ر.س | 2% | + تحليلات متقدمة |
| مؤسسي | اتفاقية | 1.5% | + API + مدير حساب |

#### 2.2.2 سياسة الأسعار
- **الشفافية:** جميع الأسعار معروضة بوضوح
- **العملة:** الريال السعودي (SAR)
- **التحديث:** أسعار المعادن تُحدث كل 15 دقيقة
- **الضريبة:** تُضاف ضريبة القيمة المضافة (15%) على الخدمات

### 2.3 سياسة الجودة

#### 2.3.1 معايير جودة المنتجات
```yaml
المعادن الثمينة (ذهب، فضة):
  - شهادة نقاء من مختبر معتمد
  - ختم الصائغ أو المصنع
  - وثيقة المنشأ

المعادن الصناعية (نحاس، ألومنيوم):
  - تحليل كيميائي للتركيبة
  - شهادة الوزن من ميزان معتمد
  - فحص بصري للجودة

السكراب والخردة:
  - تصنيف الدرجة (A, B, C)
  - نسبة الشوائب
  - صور موثقة
```

| المؤشر | الهدف | القياس |
|--------|-------|--------|
| رضا العملاء | ≥ 95% | استبيان شهري |
| دقة التوصيل | ≥ 98% | تلقائي |
| حل الشكاوى | ≤ 24 ساعة | تلقائي |
| جودة المنتجات | ≥ 99% | فحص عينات |
| توفر النظام | ≥ 99.9% | مراقبة تلقائية |

```

│  جديد   │ → │ مؤكد    │ → │ قيد     │ → │  جاهز   │

└─────────┘    └─────────┘    └─────────┘    └─────────┘
```

#### 3.1.2 الإجراءات التفصيلية

**إجراء: استلام طلب جديد**
```yaml
الرقم: PRO-ORD-001
المسؤول: نظام إدارة الطلبات
التوقيت: فوري (آلي)

الخطوات:
  1. استلام الطلب من العميل
  2. التحقق من صحة البيانات
  3. التحقق من توفر المخزون
  4. حجز الكمية مؤقتاً (30 دقيقة)
  5. إرسال إشعار للبائع
  6. انتظار تأكيد البائع

المخرجات:
  - رقم طلب فريد
  - إشعار للعميل
  - إشعار للبائع
```

**إجراء: تأكيد الطلب**
```yaml
الرقم: PRO-ORD-002
المسؤول: البائع
التوقيت: خلال 4 ساعات

الخطوات:
  1. مراجعة تفاصيل الطلب
  2. التأكد من توفر البضاعة
  3. تأكيد السعر النهائي
  4. تحديد موعد التجهيز
  5. الضغط على "تأكيد الطلب"

المخرجات:
  - تحويل حالة الطلب إلى "مؤكد"
  - إصدار فاتورة أولية
  - إشعار العميل للدفع
```

## 🔒 الفصل الرابع: سياسات الأمان والخصوصية

### 4.1 أمان المعلومات

#### 4.1.1 تصنيف البيانات
| التصنيف | الأمثلة | مستوى الحماية |
|---------|--------|---------------|
| سري جداً | بيانات الدفع، كلمات المرور | تشفير AES-256 |
| سري | معلومات العملاء الشخصية | تشفير + وصول محدود |
| داخلي | تقارير الأعمال | وصول للموظفين فقط |
| عام | معلومات المنتجات | متاح للجميع |

#### 4.1.2 سياسة كلمات المرور
```yaml
المتطلبات:
  - الحد الأدنى: 12 حرف
  - أحرف كبيرة وصغيرة
  - أرقام
  - رموز خاصة
  - تغيير كل 90 يوم
  - عدم تكرار آخر 5 كلمات

المصادقة الثنائية:
  - إلزامية للتجار
  - إلزامية للمعاملات > 10,000 ر.س
  - اختيارية للعملاء العاديين
```

#### 4.1.3 النسخ الاحتياطي
```yaml
النسخ اليومي:
  - قاعدة البيانات: كل 6 ساعات
  - الملفات: يومياً
  - الاحتفاظ: 30 يوم

النسخ الأسبوعي:
  - نسخة كاملة كل جمعة
  - الاحتفاظ: 12 أسبوع

النسخ الشهري:
  - نسخة أرشيفية
  - الاحتفاظ: 7 سنوات

الموقع:
  - نسخة محلية (الرياض)
  - نسخة بعيدة (جدة)
  - نسخة سحابية (مشفرة)
```

```
المبادئ:
1. الحلال بيّن والحرام بيّن
2. لا ضرر ولا ضرار
3. الغرم بالغنم (المخاطرة مقابل الربح)
4. الأمور بمقاصدها
5. البيّعان بالخيار ما لم يتفرقا
```
