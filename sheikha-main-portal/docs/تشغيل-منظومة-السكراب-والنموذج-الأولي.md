# تشغيل منظومة السكراب والنموذج الأولي للمنتج

## 1. التشغيل الظاهر: منظومة السكراب فقط

- **في الواجهة (عند فتح الموقع):** يظهر للمستخدم فقط **منظومة السكراب** — أفضل وأكبر منظومة ومحرك للسكراب بالكون.
- **المنتج الأول / النموذج الأولي:** بيع السكراب — دخول السوق، تسجيل تاجر، عرض بضائع، تواصل وبيع.
- **ريادة الأعمال:** رابط واضح من الصفحة الرئيسية إلى صفحة ريادة الأعمال.

## 2. باقي الأنظمة: تعمل بالخلفية

- **ERP، SCM، ريادة الأعمال (منظومة)، مسرعات الأعمال، حاضنات الأعمال، المنطق، البرمجة، إلخ** — تعمل في الخلفية ولا تُعرض في الصفحة الرئيسية كأقسام رئيسية.
- يمكن الوصول إليها عبر واجهات API أو صفحات مخصصة (مثل ريادة-الأعمال.html، نظام-البرمجة.html).

## 3. واجهات ERP و SCM

- **GET /api/erp/status** — حالة نظام ERP (مفعّل).
- **GET /api/scm/status** — حالة نظام SCM (مفعّل).

إذا ظهر لك **Cannot GET /api/erp/status** أو **Cannot GET /api/scm/status**:

1. أوقف الخادم (في الطرفية: **Ctrl+C**).
2. أعد التشغيل من مجلد البوابة:  
   `cd "/Users/salmanalrajeh/Library/CloudStorage/OneDrive-Personal/Sheikha/sheikha-main-portal"`  
   ثم: `npm start`
3. افتح من المتصفح:  
   http://localhost:8080/api/erp/status  
   http://localhost:8080/api/scm/status  

## 4. دخول السوق بالمنتج الأول (النموذج الأولي)

| الخطوة | الرابط / الإجراء |
|--------|-------------------|
| 1 | **تسجيل تاجر** — [تسجيل-تاجر.html](http://localhost:8080/تسجيل-تاجر.html) |
| 2 | **عرض بضائع (المنتج الأول: بيع السكراب)** — [عرض-بضائع.html](http://localhost:8080/عرض-بضائع.html) |
| 3 | **ريادة الأعمال** — [ريادة-الأعمال.html](http://localhost:8080/ريادة-الأعمال.html) |

الصفحة الرئيسية تعرض زرّاً واضحاً: **«دخول السوق — المنتج الأول (بيع السكراب)»** وزر **«ريادة الأعمال»**.
