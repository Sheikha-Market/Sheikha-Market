## 1. الإمكانية الكاملة لوسوم Meta

التحكم الكامل بكيفية ظهور موقع شيخة، أرشفته، وتفاعله مع المتصفحات والمنصات.

### 1.1 التحكم في الظهور بمحركات البحث (SEO)

| الوسم | الغرض | المثال |
|------|-------|--------|
| **description** | النص الجذاب تحت الرابط في جوجل — زيادة نسبة النقر | `منظومة وسوق شيخة — أول مركز اقتصادي إسلامي رقمي...` |
| **robots** | أوامر صريحة لمحركات البحث | `index, follow` أو `noindex, nofollow` للوحات التحكم |
| **keywords** | كلمات مفتاحية — محركات داخلية ومتخصصة | `شيخة، سوق المعادن، سكراب، LME، مؤشر شيخة` |
| **canonical** | الرابط الأساسي — تجنب المحتوى المكرر | `https://sheikha.top/` |

### 1.2 التوافق مع الأجهزة والمتصفحات (Technical)

| الوسم | الغرض |
|------|-------|
| **viewport** | استجابة التصميم — بدونها يظهر الموقع مشوهاً على الجوال |
| **charset** | ترميز UTF-8 — ضمان ظهور النصوص العربية صحيحة |
| **theme-color** | لون شريطة المتصفح — ذهبي #D4AF37 |
| **refresh** | توجيه تلقائي لصفحة أخرى (اختياري) |

### 1.3 التحكم في الانتشار الاجتماعي (Social Media)

| الوسم | المنصة | الغرض |
|------|--------|-------|
| **og:title** | فيسبوك، واتساب، لينكدإن | العنوان عند المشاركة |
| **og:description** | — | الوصف |
| **og:image** | — | الصورة (1200×630 PNG) |
| **og:url** | — | الرابط الأساسي |
| **og:locale** | — | ar_SA |
| **twitter:card** | X (تويتر) | summary_large_image |
| **twitter:image** | — | نفس صورة OG |

### 1.4 خصائص أمنية وتعريفية

| الوسم | الغرض |
|------|-------|
| **Content-Security-Policy** | حماية من هجمات XSS — عبر Helmet في الخادم |
| **author** | سلمان أحمد بن سلمان الراجح |
| **copyright** | © 2026 — جميع الحقوق محفوظة |

## 2. الملفات والمسارات

| الملف | الغرض |
|-------|--------|
| `data/sheikha-meta-tags-config.json` | تكوين وسوم Meta لكل صفحة |
| `public/robots.txt` | توجيه محركات البحث — Allow/Disallow |
| `public/sitemap.xml` | خريطة الموقع للمحركات |
| `public/images/og-sheikha.png` | صورة المشاركة 1200×630 |

| `docs/تكامل-Meta-وسوم-شيخة.md` | هذا التوثيق |
| `docs/تحقق-Meta-شيخة.md` | قائمة تحقق وتفعيل |

## 4. الصفحات المُفعّلة

| الصفحة | robots | og:image | canonical |
|--------|--------|----------|-----------|
| index.html | index, follow | ✓ | ✓ |
| سوق-شيخة.html | index, follow | ✓ | ✓ |
| المجتمع.html | index, follow | ✓ | ✓ |
| تسجيل-الشركات.html | index, follow | ✓ | ✓ |
| الشريعة-الاسلامية.html | index, follow | ✓ | ✓ |
| خريطة-الأرض-التفاعلية.html | index, follow | ✓ | ✓ |
| هوية-السوق.html | index, follow | ✓ | ✓ |
| سياسة-الخصوصية.html | index, follow | ✓ | ✓ |
| الشروط-والسياسات.html | index, follow | ✓ | ✓ |
| فهرس-المنظومة.html | index, follow | ✓ | ✓ |
| تعليم-المجتمع.html | index, follow | ✓ | ✓ |
| لوحة-الادمن | noindex, nofollow | — | — |

**جميع الصفحات أعلاه (ما عدا لوحة-الادمن) مُفعّلة بـ:** `sheikha-meta-injector.js` + `sheikha-meta-helper.js`

### حقن ديناميكي
`/js/sheikha-meta-injector.js` — يحمل التكوين من `/api/meta/config` ويحقن الوسوم حسب مسار الصفحة. للصفحات الجديدة بدون وسوم ثابتة:

```html
<script src="/js/sheikha-meta-injector.js" defer></script>
```

### مساعد التحقق
`/js/sheikha-meta-helper.js` — يتحقق تلقائياً عند تحميل الصفحة من وجود وسوم Meta الأساسية ويُبلغ في الكونسول عن النقص. لا يحتاج استدعاء يدوي.

```html
<script src="/js/sheikha-meta-helper.js"></script>
```
