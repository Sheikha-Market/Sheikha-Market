## ٢. تشغيل الموقع وجذب المستخدمين برضاهم

- **تشغيل الموقع:** الخادم يعمل على المنفذ 8080؛ أكثر من ٢٠٠ API؛ PWA مفعّل.
- **جذب المستخدمين برضاهم:** لا إكراه — تسجيل اختياري؛ مصادقة متعددة (نفاذ، جوجل، مايكروسوفت، أبل، بريد/جوال)؛ شفافية في الشريعة والسياسات.
- **بأمر الله:** المبادئ من الكتاب والسنة؛ لا ربا ولا غرر ولا غش؛ مؤشر شيخة للمعرفة فقط.

## ٥. أفضل التقنيات بدون ضرر — بالكتاب والسنة

- **تفعيل مؤكد:** أمان ٩ طبقات؛ Lazy Loading للمحركات؛ ميثاق شيخة مُفعّل.
- **قوة تقنية وتجارية ورقمية وحوسبية:** محركات السوق، الشريعة، العاصمة الاقتصادية، رياض الجنة، الترجمة، OAuth، JWT، WebSocket، PWA.
- **بدون ضرر ولا ضرار:** لا كشف أسرار؛ لا قرار مالي أو فتوى من الآلة؛ الترجمة والـ API ترجع النص الأصلي عند الفشل؛ واجهات واضحة عند عدم إعداد مزود (مثل جوجل).

## ٦. خطوات تفعيل الدخول بجوجل (للمشغّل)

1. انسخ `.env.example` إلى `.env`.
2. من Google Cloud Console أنشئ OAuth 2.0 Client ID (ويب).
3. ضع في `.env`: `GOOGLE_CLIENT_ID=...` و `GOOGLE_CLIENT_SECRET=...`.
4. أضف في Console في «Authorized redirect URIs»:
   - إنتاج: `https://www.sheikha.top/api/auth/google/callback`
   - تطوير: `http://localhost:8080/api/auth/google/callback`
5. أعد تشغيل الخادم.
