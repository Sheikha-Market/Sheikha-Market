| الخدمة | الوصف | السعر الشهري |

| 📑 المطبوعات | بطاقات، ترويسة، فواتير | 800 ريال (مرة واحدة) |

| الباقة | المميزات | السعر الشهري |

| 🌐 احترافية | متجر كامل، دفع إلكتروني | 799 ريال |
| 🌐 مؤسسية | تخصيص كامل، دعم 24/7 | 1,499 ريال |

## 🔗 خريطة APIs

```
/api/
├── /companies/
│   ├── POST /register          # تسجيل شركة
│   ├── GET /registered         # قائمة الشركات
│   └── GET /registered/:id     # تفاصيل شركة
│
├── /company/:id/
│   ├── GET /products           # المنتجات
│   ├── POST /products          # إضافة منتج
│   ├── GET /analytics          # التحليلات
│   ├── POST /ai/chat           # ذكاء شيخة
│   └── GET /store              # المتجر
│
├── /marketing/
│   ├── GET /packages           # باقات التسويق
│   ├── POST /branding          # طلب هوية
│   ├── POST /website           # طلب موقع
│   └── POST /emails            # طلب إيميلات
│
├── /ai/
│   ├── POST /chat              # محادثة شيخة
│   └── GET /capabilities       # قدرات الذكاء
│
└── /network/
    └── GET /unified-goals      # الأهداف الموحدة
```
