## 2) التعاريف
### المعلومات السرية تشمل:
- بنية المنظومة التقنية والمعمارية لمنصة شيخة.
- أكواد المصدر، APIs، مفاتيح التشفير، وبيانات الاعتماد.
- خطط العمل، النماذج المالية، استراتيجيات التسعير.
- بيانات المستخدمين والتجار والشركات المسجلة.
- تقارير الأداء، مؤشرات KPIs، وتحليلات السوق.
- المعلومات المتعلقة بالشراكة السعودية-الأمريكية في المواد النادرة.

### الغرض المصرح:
التكامل التقني بين منظومة شيخة (سوق إسلامي رقمي للمعادن والسكراب) وخدمات Google Cloud Platform التالية:
- Vertex AI (التدقيق الشرعي الآلي والذكاء الاصطناعي)
- Cloud Run (استضافة الخدمات بدون خادم)
- Cloud Firestore (قاعدة بيانات لحظية)
- Cloud Storage (تخزين الوثائق والمواد)
- Cloud Functions (وظائف سحابية آلية)
- Secret Manager (إدارة المفاتيح والأسرار)

## 3) التزامات الطرف المستلم
يلتزم كل طرف بما يلي:
1. استخدام المعلومات السرية **فقط** للغرض المصرح أعلاه.
2. عدم نسخ أو نشر أو مشاركة المعلومات السرية دون موافقة خطية مسبقة.
3. قصر الوصول على الموظفين والمقاولين الذين يحتاجون المعرفة فعليًا.
4. تطبيق معايير حماية تقنية لا تقل عن:
   - TLS 1.3 للنقل
   - AES-256 للتخزين
   - MFA للوصول الإداري
   - سجلات تدقيق شاملة
5. إبلاغ الطرف الآخر فورًا (خلال 24 ساعة) عند أي حادث أمني أو إفصاح غير مصرح.

## 4) الاستثناءات
لا تشمل السرية المعلومات التي:
- كانت متاحة للعامة قبل الإفصاح أو أصبحت عامة دون خرق للاتفاقية.
- كانت بحوزة المستلم قبل الإفصاح ومثبتة بسجلات موثقة.
- تم تطويرها بشكل مستقل دون استخدام المعلومات السرية.
- يُطلب الإفصاح عنها بموجب أمر قضائي أو نظامي، مع إشعار مسبق للطرف الآخر قدر الإمكان.

## 5) الملكية الفكرية
- جميع الحقوق في المعلومات السرية تظل مملوكة للطرف المفصح.
- لا تمنح هذه الاتفاقية أي ترخيص ضمني أو صريح لاستخدام براءات الاختراع أو العلامات التجارية أو حقوق النشر.
- المخرجات المشتركة (إن وجدت) تخضع لاتفاقية منفصلة.

## 6) مدة السرية
- **مدة الاتفاقية:** 24 شهرًا من تاريخ التوقيع.
- **التزام السرية:** يستمر لمدة **5 سنوات** بعد انتهاء الاتفاقية أو حتى تصبح المعلومات عامة بشكل مشروع.

## 7) معالجة البيانات والامتثال
### ضوابط شرعية (إسلامية):
- لا ربا: لا تستخدم البيانات في معاملات ربوية.
- لا غرر: شفافية كاملة في استخدام البيانات.
- لا بيع بيانات: لا يجوز بيع أو تأجير البيانات الشخصية.

### ضوابط تقنية:
- التشفير الإلزامي عند النقل والتخزين.
- سجلات تدقيق شاملة لكل وصول لبيانات حساسة.
- منع تخزين الأسرار في مستودعات عامة.
- تطبيق مبدأ أقل صلاحية (Least Privilege).

### إعادة/إتلاف البيانات:
عند انتهاء العلاقة أو بناءً على طلب الطرف المفصح:
- إعادة جميع المواد السرية.
- أو إتلافها بشكل آمن مع تقديم شهادة إتلاف.

## 8) عدم الاستقطاب
