# 🍎 التطوير على Mac (MacBook/iMac)

## الإعداد الأولي

### 1️⃣ المتطلبات الأساسية:

```bash
# التحقق من Node.js
node --version    # يجب أن يكون v18+ أو v20+
npm --version     # يجب أن يكون v9+

# إذا لم يكن مثبتاً:
brew install node

# أو استخدم النسخة المثبتة مسبقاً:
/usr/local/bin/node --version
```

### 2️⃣ الوصول للمشروع:

```bash
# المسار الكامل للمشروع
cd "/Users/salmanalrajeh/Library/CloudStorage/OneDrive-Personal/Sheikha/sheikha-main-portal"

# أو باختصار (إنشاء alias)
echo 'alias sheikha="cd /Users/salmanalrajeh/Library/CloudStorage/OneDrive-Personal/Sheikha/sheikha-main-portal"' >> ~/.zshrc
source ~/.zshrc

# الآن يمكنك استخدام:
sheikha
```

### 3️⃣ تثبيت التبعيات:

```bash
# تثبيت جميع التبعيات
npm install

# أو إذا واجهت مشاكل:
npm install --legacy-peer-deps
```

### 4️⃣ إعداد ملف البيئة (.env):

```bash
# إنشاء/تعديل ملف .env
nano .env

# المحتوى المطلوب:
PORT=8080
NODE_ENV=development
JWT_SECRET=sheikha-jwt-secret-change-in-production

# للذكاء الاصطناعي (اختياري):
OPENAI_API_KEY=sk-your-key-here
AI_PROVIDER=openai
AI_LLM_MODEL=gpt-4o-mini
```

### 5️⃣ تشغيل الخادم:

```bash
# الطريقة 1: باستخدام npm
npm start

# الطريقة 2: مباشرة (إذا واجهت مشاكل NVM)
/usr/local/bin/node server.js

# الطريقة 3: مع إعادة التشغيل التلقائي
npx nodemon server.js
```

### 6️⃣ الوصول للموقع:

```
🌐 الموقع المحلي: http://localhost:8080
📊 API Docs: http://localhost:8080/api-docs.html
🔧 DevOps: http://localhost:8080/devops.html
🤖 المساعد: http://localhost:8080/مساعد-شيخة.html
```

## سكريبت التشغيل السريع (Mac):

```bash
#!/bin/bash
# حفظ كـ: start-sheikha.sh

echo "🚀 بدء تشغيل منظومة شيخة..."

# الانتقال للمجلد
cd "/Users/salmanalrajeh/Library/CloudStorage/OneDrive-Personal/Sheikha/sheikha-main-portal"

# إيقاف أي خادم سابق
lsof -ti:8080 | xargs kill -9 2>/dev/null

# تشغيل الخادم
echo "⚡ تشغيل الخادم..."
/usr/local/bin/node server.js &

# انتظار بدء الخادم
sleep 3

# فتح المتصفح
echo "🌐 فتح المتصفح..."
open http://localhost:8080

echo "✅ منظومة شيخة جاهزة!"
echo "📍 الموقع: http://localhost:8080"
```

```bash
# جعل السكريبت قابل للتنفيذ
chmod +x start-sheikha.sh

# التشغيل
./start-sheikha.sh
```

# 🎮 التطوير على Windows (MSI/أي جهاز Windows)

## الإعداد الأولي

### 1️⃣ المتطلبات الأساسية:

```powershell
# التحقق من Node.js
node --version
npm --version

# إذا لم يكن مثبتاً، حمّل من:
# https://nodejs.org/en/download/
# أو باستخدام Chocolatey:
choco install nodejs
```

### 2️⃣ الوصول للمشروع:

```powershell
# إذا كان OneDrive مزامناً:
cd "C:\Users\YourUsername\OneDrive\Sheikha\sheikha-main-portal"

# أو استنساخ من Git (إذا متاح):
git clone https://github.com/your-repo/sheikha-main-portal.git
cd sheikha-main-portal
```

### 3️⃣ نقل المشروع من Mac إلى Windows:

**الطريقة 1: عبر OneDrive (الأسهل)**
```
✅ المشروع موجود في OneDrive
✅ يتزامن تلقائياً بين الأجهزة
📍 المسار: OneDrive/Sheikha/sheikha-main-portal
```

**الطريقة 2: عبر USB/نقل مباشر**
```powershell
# نسخ المجلد بالكامل
xcopy /E /I "D:\sheikha-main-portal" "C:\Projects\sheikha-main-portal"
```

**الطريقة 3: عبر Git**
```powershell
# إذا كان المشروع على GitHub
git clone https://github.com/your-username/sheikha-main-portal.git
```

### 4️⃣ تثبيت التبعيات:

```powershell
# الانتقال للمجلد
cd sheikha-main-portal

# تثبيت التبعيات
npm install
```

### 5️⃣ إعداد ملف البيئة:

```powershell
# إنشاء ملف .env
notepad .env

# أضف المحتوى:
PORT=8080
NODE_ENV=development
JWT_SECRET=sheikha-jwt-secret-change-in-production
OPENAI_API_KEY=sk-your-key-here
```

### 6️⃣ تشغيل الخادم:

```powershell
# تشغيل الخادم
npm start

# أو مباشرة
node server.js

# مع إعادة التشغيل التلقائي
npx nodemon server.js
```

## سكريبت التشغيل السريع (Windows):

```batch
@echo off
REM حفظ كـ: start-sheikha.bat

echo 🚀 بدء تشغيل منظومة شيخة...

REM الانتقال للمجلد
cd /d "C:\Users\%USERNAME%\OneDrive\Sheikha\sheikha-main-portal"

REM إيقاف أي خادم سابق على المنفذ 8080
for /f "tokens=5" %%a in ('netstat -aon ^| find ":8080"') do taskkill /F /PID %%a 2>nul

REM تشغيل الخادم
echo ⚡ تشغيل الخادم...
start /B node server.js

REM انتظار
timeout /t 3 /nobreak > nul

REM فتح المتصفح
echo 🌐 فتح المتصفح...
start http://localhost:8080

echo ✅ منظومة شيخة جاهزة!
echo 📍 الموقع: http://localhost:8080
pause
```
