## الهدف الأعلى — إن شاء الله

- **تحقيق إن شاء الله الأعلى قيمة والأفضل إن شاء الله عبادة لله** — المنظومة تُبنى لتحقيق أعلى قيمة للمستخدمين والبيئة والمجتمع، والأفضل في التخصص، في إطار العبادة لله (حلال، أمانة، شفافية، بلا ضرر ولا ضرار).
- **التشغيل الآن** — تفعيل الموقع والخادم والخدمات فوراً وفق الخطوات أدناه.

# الجزء الثاني — تخطيط الموقع لتحقيق الأهداف

## 2.1 ربط الصفحات بالأهداف

| الهدف | أين يظهر في الموقع |
|--------|----------------------|
| **بلا ضرر ولا ضرار** | الرئيسية (قسم الأهداف)؛ الفوتر؛ صفحة الهوية والبيئة والجودة. |
| **حب العلامة وأمان** | الرئيسية؛ الفوتر؛ كل صفحة (هوية موحدة). |
| **محافظة على البيئة والصحة والبشر** | الرئيسية؛ الفوتر؛ الهوية والبيئة والجودة؛ بيان إعادة التدوير. |

## 2.2 مخطط الموقع لتحقيق الأهداف

| الصفحة | المسار | الغرض لتحقيق الأهداف |
|--------|--------|------------------------|
| **الرئيسية** | index.html | البوابة؛ الأهداف الثلاثة واضحة؛ دخول السوق (تسجيل، عرض، لوحة تحكم). |
| **تسجيل متجر** | تسجيل-تاجر.html | فتح متجر — تحقيق «دخول السوق» و«أمان» عبر تسجيل موحد. |
| **عرض بضائع** | عرض-بضائع.html | إضافة عروض — تحقيق الشفافية والجودة عبر تصنيف وعرض موحد. |
| **لوحة التحكم** | لوحة-التحكم.html | إدارة المتجر والمنتجات والمؤشرات — تحقيق القياس والأمان. |
| **خريطة الموقع** | خريطة-الموقع.html | روابط كل الصفحات — وضوح ووصول لجميع الأهداف الداعمة. |
| **الهوية والبيئة والجودة** | الهوية-والبيئة-والجودة.html | الأهداف الأخلاقية والجامع؛ الهوية والعلامة؛ البيئة. |

**النتيجة:** تخطيط الموقع مصمم بحيث كل هدف له مكان واضح في الصفحات الأساسية والفوتر؛ المراجعة عند إضافة/حذف صفحات — انظر: مخطط-الموقع-والنظام-المتكامل.md.

# الجزء الخامس — التشغيل الآن

## 5.1 متطلبات التشغيل

| المتطلب | الوصف |
|--------|--------|
| **Node.js** | إصدار 18 أو 20 موصى به (لتجنب أخطاء على بعض الأنظمة). |
| **المجلد** | الانتقال إلى مجلد المشروع: `sheikha-main-portal`. |
| **التبعيات** | تثبيت مرة واحدة: `npm install`. |

## 5.2 خطوات التشغيل الفورية

| الخطوة | الأمر أو الإجراء |
|--------|-------------------|
| **1** | افتح الطرفية (Terminal). |
| **2** | انتقل لمجلد المشروع: `cd "المسار_الكامل_إلى/sheikha-main-portal"` (أو استخدم المسار الفعلي لمشروعك). |
| **3** | (اختياري) تفعيل nvm واستخدام Node 20: `nvm use 20` أو `nvm install 20 && nvm use 20`. |
| **4** | تثبيت التبعيات إن لم تكن مُثبتة: `npm install`. |
| **5** | تشغيل الخادم: `npm start`. |
| **6** | عند ظهور **Server running on port 8080** (أو المنفذ المعرّف في .env)، افتح المتصفح: **http://localhost:8080**. |

## 5.3 روابط بعد التشغيل

| الصفحة | الرابط |
|--------|--------|
| الرئيسية | http://localhost:8080 |
| تسجيل متجر | http://localhost:8080/تسجيل-تاجر.html |
| عرض بضائع | http://localhost:8080/عرض-بضائع.html |
| تسجيل الدخول | http://localhost:8080/تسجيل-الدخول.html |
| لوحة التحكم | http://localhost:8080/لوحة-التحكم.html |
| خريطة الموقع | http://localhost:8080/خريطة-الموقع.html |
| الهوية والبيئة والجودة | http://localhost:8080/الهوية-والبيئة-والجودة.html |

**إيقاف الخادم:** في الطرفية اضغط `Ctrl+C`.

**تفاصيل إضافية:** انظر الملف في جذر المشروع: **تشغيل-الآن-والبدء-بالتجارة.md** — خطوات تفصيلية (بما فيها nvm على macOS) والبدء بالتجارة.
