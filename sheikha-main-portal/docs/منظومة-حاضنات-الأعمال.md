# منظومة حاضنات الأعمال — احتضان أعمال تجارية والعمل بالخلفية

## 1) المفهوم

**حاضنات الأعمال** في سوق شيخه تعني: منظومة لاحتضان أعمال تجارية — من الفكرة أو الانطلاق إلى التفعيل وأول صفقة ثم التوسع. المنظومة تعمل **بالخلفية** لتوفير البيئة والدعم: منصة واحدة، مراحل واضحة، استشارات، لوجستيات، عقود، ومسرعات أعمال — دون أن يضطر رائد الأعمال لإدارة كل شيء يدوياً.

---

## 2) ما توفره الحاضنات (بالخلفية)

| العنصر | الوظيفة (بالخلفية) |
|--------|---------------------|
| **المنصة** | سوق شيخه — تداول، عروض، طلبات، أسعار حية — متاحة دوماً بالخلفية. |
| **المراحل** | مراحل الريادة الست — الانطلاق، التفعيل، أول صفقة، التوسع، الريادة، التجارة المتكاملة — مُطبَّقة ومنظمة بالخلفية. |
| **الاستشارات** | قناة استشارات — طلبات واستجابة — تعمل بالخلفية لدعم القرار. |
| **اللوجستيات** | ربط مع النقل والتتبع — بالخلفية — لدعم الشحن والتسليم. |
| **العقود** | إطار عقود توريد وشراء/بيع — بالخلفية — لدعم الصفقات. |
| **مسرعات الأعمال** | نظام مسرعات الأعمال (تسريع الأعمال) يعمل بالخلفية — مكمّل للحاضنات. |

---

## 3) احتضان أعمال تجارية

- **احتضان:** توفير البيئة (منصة، مراحل، خدمات، عقود) لبدء وتنمية أعمال تجارية في المعادن والسكراب.
- **أعمال تجارية:** أفراد، مؤسسات، شركات صغيرة ومتوسطة — بائع، مشترٍ، وسيط، مصنع، معيد تدوير.
- **العمل بالخلفية:** المنظومة تعمل دون توقف — الدخول والتسجيل والعروض والصفقات مدعومة بالخلفية.

المنظومة مُفعّلة وتعمل بالخلفية ضمن سوق شيخه.
