# آلية تجربة العميل — نظام آلي لتجربة المحتوى كاملاً

## 1) الهدف

نظام آلي يُمكّن العميل من **تجربة المحتوى كاملاً** في سوق شيخه ([www.sheikha.top](http://www.sheikha.top)) لتحقيق الأهداف: التعريف بالمنصة، الخدمات، السوق، الخرائط والمحاكاة، متخصص السكراب، والتواصل.

---

## 2) مكوّنان للتجربة

### أ) صفحة «تجربة العميل»

- **الرابط:** `/تجربة-العميل.html`
- **المحتوى:** خطوات مرقّمة (1–6): الرئيسية، خدماتنا المتكاملة، أسعار السوق، نظام الخرائط والمحاكاة، متخصص السكراب، تواصل معنا.
- **الآلية:** كل خطوة تحتوي على وصف قصير وزر «انتقل إلى القسم» يفتح القسم أو الصفحة المناسبة. عند زيارة كل قسم تُحفظ الخطوة (محلياً) وتُظهَر كمنجزة؛ عند إكمال الجميع تظهر رسالة «أتممت تجربة المحتوى».

### ب) جولة تجربة المحتوى (على الصفحة الرئيسية)

- **الموقع:** زر «تجربة المحتوى» ثابت أسفل يسار الصفحة الرئيسية.
- **الآلية:** عند الضغط تبدأ جولة تلقائية: تظهر نافذة توضح القسم الحالي (الرئيسية، ثم الخدمات، ثم السوق، ثم الخرائط والمحاكاة، ثم تواصل معنا) مع تمرير الصفحة إلى القسم، وزرّي «التالي» و«إنهاء». عند «التالي» تنتقل الجولة للقسم التالي؛ عند «إنهاء» أو انتهاء الخطوات تُغلق النافذة.

---

## 3) تناسق المحتوى مع الأهداف

| الخطوة | الهدف |
|--------|--------|
| الرئيسية | تعريف المنصة وشريكك الموثوق وأسعار اليوم. |
| خدماتنا المتكاملة | عرض التداول، الاستشارات، اللوجستيات، إعادة التدوير، المختبرات، تحليل السوق. |
| أسعار السوق | عرض أسعار حية للمعادن والسكراب (عالمي وسعودي). |
| نظام الخرائط والمحاكاة | تتبع البضائع وتشغيل أنظمة الفضاء لتحقيق الأهداف. |
| متخصص السكراب | بحوث وتطوير وتجارة ومجتمع متخصصي السكراب. |
| تواصل معنا | روابط سريعة، خدماتنا، الأنظمة، وبدء الرحلة. |

---

## 4) الروابط في الموقع

- **القائمة العلوية:** رابط «تجربة العميل» يوجّه إلى `/تجربة-العميل.html`.
- **الفوتر:** تحت «روابط سريعة» رابط «تجربة العميل» إلى نفس الصفحة.
- **الزر الثابت:** «تجربة المحتوى» على الصفحة الرئيسية يبدأ الجولة الآلية.

تم إعداد هذه الآلية لتحقيق تجربة عميل متكاملة لمحتوى المنصة.
