# إدارة حاويات السكراب وسكراب تشليح السيارات

## 1. سكراب تشليح السيارات

تم إضافة تصنيف **سكراب تشليح السيارات** (HS 8708) في النظام:

| البند | الوصف |
|-------|--------|
| **المعرف** | 8708 |
| **الاسم بالعربي** | سكراب تشليح السيارات |
| **الاسم بالإنجليزي** | Automotive salvage / Vehicle dismantling scrap |
| **كلمات مفتاحية** | تشليح، سيارات، مركبات، قطع غيار، خردة سيارات، سكراب سيارات |

- يظهر في **تصنيفات HS** (`/api/hs-scrap-categories`) وفي **أنواع عرض السكراب** (`/api/scrap-display-types`) تحت اسم «سكراب تشليح السيارات».
- يمكن للمستخدمين إضافة منتجات من هذا النوع (سكراب تشليح سيارات) وفلترة السوق حسبه (`?scrapType=automotive_salvage` أو حسب الفئة 8708).

## 2. إدارة حاويات السكراب

### أنواع الحاويات

| النوع | الاسم بالعربي | السعة |
|-------|---------------|--------|
| small | حاوية صغيرة | 1–5 طن |
| medium | حاوية متوسطة | 5–15 طن |
| large | حاوية كبيرة | 15–25 طن |
| roll_off | حاوية رول أوف | حسب الطلب |
| open_top | حاوية مفتوحة | 20–40 قدم |

### حالات الحاوية

| الحالة | الاسم بالعربي |
|--------|---------------|
| available | متاحة |
| in_use | قيد الاستخدام |
| loaded | معبأة |
| reserved | محجوزة |
| maintenance | صيانة |

### واجهات API

| الطريقة | المسار | الوظيفة |
|---------|--------|----------|
| GET | /api/container-types | قائمة أنواع الحاويات وحالاتها |
| GET | /api/containers | قائمة الحاويات (اختياري: ?ownerId= أو ?traderId=) |
| GET | /api/my-containers | حاوياتي (يتطلب تسجيل دخول) |
| POST | /api/containers | إضافة حاوية (يتطلب تسجيل دخول) — الحقول: type، label، locationAddress، locationLat، locationLng، capacity، status |
| PUT | /api/containers/:id | تعديل حاوية (مالك فقط) |
| DELETE | /api/containers/:id | حذف حاوية (مالك فقط) |

### تخزين البيانات

- الحاويات تُحفظ في `data/containers.json`.
- كل حاوية تحتوي على: id، ownerId، traderId، storeName، storeRegion، type، label، locationAddress، locationLat، locationLng، capacity، status، createdAt.
