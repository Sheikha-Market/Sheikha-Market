## 🎯 الأهداف

### الهدف الرئيسي
توفير منظومة نقل متكاملة وآمنة للمعادن تربط بين الناقلين والتجار والمصانع بكفاءة عالية

### الأهداف الفرعية
1. **السلامة** - ضمان نقل آمن للمعادن بجميع أنواعها
2. **التتبع** - تتبع حي للشحنات GPS
3. **الكفاءة** - تقليل وقت وتكلفة الشحن
4. **الموثوقية** - شبكة ناقلين معتمدين وموثوقين
5. **التكامل** - ربط سلس مع أنظمة التجار والمصانع

## 📝 الوصف التفصيلي

### ما هي الخدمة؟
خدمة النقل والشحن توفر منصة متكاملة لنقل المعادن من أي نقطة إلى أخرى داخل المملكة وخارجها، مع توفير خيارات متعددة من الناقلين المعتمدين، وتتبع حي للشحنات، وتأمين شامل.

### أنواع الخدمة

#### أ. للتجار والمصانع (طالبي الشحن)
- طلب شحن مباشر
- مقارنة أسعار الناقلين
- اختيار نوع الشاحنة المناسب
- تتبع الشحنة
- التأمين على البضائع

#### ب. للناقلين (مقدمي الخدمة)
- استقبال طلبات شحن
- إدارة أسطول الشاحنات
- نظام GPS للتتبع
- محفظة للأرباح
- تقارير الأداء

├── عنوان الاستلام
├── عنوان التسليم

├── اختيار نوع الشاحنة

├── عرض الناقلين المتاحين
├── مقارنة الأسعار

└── المدة: 2-3 دقائق

الخطوة 4: تأكيد الطلب والدفع
├── اختيار الناقل
├── مراجعة التفاصيل النهائية

├── تتبع حي GPS

الخطوة 6: استلام وتقييم
├── تأكيد الاستلام
├── فحص البضاعة
├── تقييم الناقل
└── المدة: 5 دقائق
```

### ب. إجراءات التسجيل كناقل

```
المرحلة 1: تقديم الطلب

├── تحديد مناطق العمل

### ج. إجراءات تنفيذ الشحنة (للناقلين)

```
1. استلام طلب شحن
   ├── إشعار بطلب جديد
   ├── مراجعة التفاصيل
   └── قبول أو رفض الطلب
   
2. تأكيد الموعد
   ├── التواصل مع العميل
   ├── تأكيد موعد الاستلام
   └── تحديث الحالة في النظام

3. الاستلام
   ├── الوصول لموقع الاستلام
   ├── فحص البضاعة
   ├── توقيع سند الاستلام
   ├── تحميل البضاعة
   └── تحديث: "جاري النقل"

4. النقل
   ├── تشغيل GPS
   ├── الالتزام بالمسار
   ├── إبلاغ عن أي طوارئ
   └── الحالة: "في الطريق"

5. التسليم
   ├── الوصول لموقع التسليم
   ├── تفريغ البضاعة
   ├── توقيع سند التسليم
   ├── تحديث: "تم التسليم"
   └── رفع صور التسليم

6. استلام الأرباح
   ├── تحويل للمحفظة
   ├── خصم عمولة المنصة (8%)
   └── متاح للسحب
```

## ☪️ الالتزام الشرعي

1. **الأمانة** - الحفاظ على البضاعة المؤتمنة
2. **الصدق** - الالتزام بالمواعيد والأسعار المعلنة
3. **عدم الغش** - تسليم البضاعة كما استُلمت
4. **الوفاء بالعقد** - تنفيذ الشحن كما تم الاتفاق
5. **لا ضرر** - تعويض أي ضرر بالإهمال
