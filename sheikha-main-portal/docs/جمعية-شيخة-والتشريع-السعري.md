### الرؤية:
> **"أن تكون جمعية شيخة المرجع العالمي الأول لتحقيق العدالة السعرية في تجارة المعادن والسكراب"**

### الرسالة:
> تأسيس منظومة عادلة وشفافة لتسعير المعادن والسكراب، قائمة على مبادئ الشريعة الإسلامية، تحقق المنفعة العامة لجميع الأطراف دون ضرر أو ضرار.

### القيم الأساسية:
1. ⚖️ **العدالة** - تسعير عادل لا يظلم أحداً
2. 🔍 **الشفافية** - كل الأسعار والمعادلات معلنة
3. 🤝 **التراضي** - تجارة عن تراض من الجميع
4. 🌿 **الاستدامة** - الحفاظ على البيئة والموارد
5. 📖 **الشرعية** - الالتزام بأحكام الشريعة الإسلامية

## 👥 فئات العضوية

### 1️⃣ العضوية المؤسسة (Founding Members):
- المؤسسون الأوائل للجمعية
- حق التصويت الكامل
- عضوية مدى الحياة
- **الرسوم:** 50,000 ر.س (مرة واحدة)

### 2️⃣ العضوية الذهبية (Gold Members):
- الشركات والمصانع الكبرى
- حق التصويت الكامل
- أولوية في الخدمات
- **الرسوم:** 25,000 ر.س/سنة

### 3️⃣ العضوية الفضية (Silver Members):
- التجار والموزعون
- حق التصويت
- خدمات متقدمة
- **الرسوم:** 10,000 ر.س/سنة

### 4️⃣ العضوية البرونزية (Bronze Members):
- صغار التجار والمستثمرون
- حق حضور الاجتماعات
- خدمات أساسية
- **الرسوم:** 2,500 ر.س/سنة

### 5️⃣ العضوية الرقمية (Digital Members):
- الأفراد والمهتمون
- وصول للمنصة والبيانات
- **الرسوم:** 500 ر.س/سنة أو مجانية

## 📌 الثوابت الأساسية (لا تتغير أبداً)

### 1️⃣ ثابت تكلفة الإنتاج الأساسية (PCF - Production Cost Factor):

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                     ثابت تكلفة الإنتاج الأساسية (PCF)                               │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  PCF = تكلفة المواد الخام الأساسية + تكلفة الطاقة الأساسية + تكلفة العمالة الأساسية │
│                                                                                     │
│  هذا الثابت يمثل الحد الأدنى المطلق لتكلفة الإنتاج                                  │
│  ولا يمكن بيع أي منتج بأقل منه (لضمان عدم الإضرار بالمنتج)                         │
│                                                                                     │
│  مثال للحديد:                                                                       │
│  PCF(حديد) = 1,200 ر.س/طن (خام الحديد) + 400 ر.س/طن (طاقة) + 200 ر.س/طن (عمالة)    │
│  PCF(حديد) = 1,800 ر.س/طن (الحد الأدنى الثابت)                                      │
│                                                                                     │
└─────────────────────────────────────────────────────────────────────────────────────┘
```

### 2️⃣ ثابت هامش الربح العادل (FPM - Fair Profit Margin):

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                     ثابت هامش الربح العادل (FPM)                                    │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  FPM = نسبة ربح ثابتة تضمن استمرارية العمل دون جشع                                 │
│                                                                                     │
│  ┌────────────────────────┬──────────────┬──────────────────────────────────────┐  │
│  │ نوع المنتج             │ FPM الثابت   │ المبرر الشرعي والاقتصادي             │  │
│  ├────────────────────────┼──────────────┼──────────────────────────────────────┤  │
│  │ المعادن الأساسية       │ 8% - 12%     │ سلع ضرورية، هامش محدود              │  │
│  │ المعادن الثمينة        │ 2% - 5%      │ قيمة عالية، هامش منخفض              │  │
│  │ السكراب               │ 10% - 15%    │ جهد التجميع والفرز                   │  │
│  │ الخدمات اللوجستية      │ 12% - 18%    │ تكاليف تشغيلية                       │  │
│  │ الخدمات المضافة        │ 15% - 25%    │ قيمة مضافة                           │  │
│  └────────────────────────┴──────────────┴──────────────────────────────────────┘  │
│                                                                                     │
│  📖 الأساس الشرعي: "رحم الله امرأً سمحاً إذا باع، سمحاً إذا اشترى"                 │
│                                                                                     │
└─────────────────────────────────────────────────────────────────────────────────────┘
```

### 3️⃣ ثابت حماية المستهلك (CPF - Consumer Protection Factor):

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                     ثابت حماية المستهلك (CPF)                                       │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  CPF = الحد الأقصى للزيادة السعرية المسموحة في حالات الطوارئ                        │
│                                                                                     │
│  CPF الثابت = 25% (الحد الأقصى للزيادة عن السعر المرجعي)                           │
│                                                                                     │
│  أي سعر يتجاوز: السعر المرجعي × 1.25 يُعتبر احتكاراً ويُرفض                        │
│                                                                                     │
│  📖 الأساس الشرعي: "لا يحتكر إلا خاطئ"                                              │
│                                                                                     │
└─────────────────────────────────────────────────────────────────────────────────────┘
```

### 4️⃣ ثابت حماية المنتج (PPF - Producer Protection Factor):

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                     ثابت حماية المنتج (PPF)                                         │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  PPF = الحد الأدنى للسعر الذي يضمن عدم خسارة المنتج                                │
│                                                                                     │
│  PPF الثابت = تكلفة الإنتاج + 5% (الحد الأدنى للربح)                               │
│                                                                                     │
│  أي سعر أقل من: تكلفة الإنتاج × 1.05 يُعتبر إغراقاً ويُرفض                         │
│                                                                                     │
│  📖 الأساس الشرعي: "لا ضرر ولا ضرار"                                                │
│                                                                                     │
└─────────────────────────────────────────────────────────────────────────────────────┘
```

## 📈 المتغيرات اللحظية (تتغير يومياً)

### 1️⃣ متغير العرض والطلب (SDV - Supply-Demand Variable):

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                     متغير العرض والطلب (SDV)                                        │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  SDV = (الطلب الفعلي / العرض المتاح) × معامل التوازن                               │
│                                                                                     │
│  ┌────────────────────────┬──────────────┬──────────────────────────────────────┐  │
│  │ الحالة                 │ SDV          │ التأثير على السعر                    │  │
│  ├────────────────────────┼──────────────┼──────────────────────────────────────┤  │
│  │ فائض عرض كبير         │ 0.85 - 0.95  │ انخفاض حتى PPF                       │  │
│  │ توازن طبيعي           │ 0.95 - 1.05  │ سعر مرجعي                            │  │
│  │ طلب زائد متوسط        │ 1.05 - 1.15  │ زيادة معتدلة                         │  │
│  │ طلب زائد كبير         │ 1.15 - 1.25  │ زيادة (لا تتجاوز CPF)                │  │
│  └────────────────────────┴──────────────┴──────────────────────────────────────┘  │
│                                                                                     │
│  📌 SDV محدود بين 0.85 و 1.25 لضمان العدالة                                        │
│                                                                                     │
└─────────────────────────────────────────────────────────────────────────────────────┘
```

### 2️⃣ متغير الأسعار العالمية (GPV - Global Price Variable):

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                     متغير الأسعار العالمية (GPV)                                    │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  GPV = (سعر LME الحالي / سعر LME المرجعي) × وزن التأثير                            │
│                                                                                     │
│  وزن تأثير الأسعار العالمية:                                                       │
│  • للمعادن المستوردة: 70%                                                          │
│  • للمعادن المحلية: 40%                                                            │
│  • للسكراب: 30%                                                                    │
│                                                                                     │
│  📌 GPV محدود بين 0.80 و 1.30 لحماية السوق المحلي                                  │
│                                                                                     │
└─────────────────────────────────────────────────────────────────────────────────────┘
```

## 🔢 المعادلة الكاملة للتسعير العادل

```
╔══════════════════════════════════════════════════════════════════════════════════════╗
║                                                                                      ║
║              🔢 معادلة شيخة الشاملة للتسعير العادل                                   ║
║                                                                                      ║
║  ┌──────────────────────────────────────────────────────────────────────────────┐   ║
║  │                                                                              │   ║
║  │   السعر العادل = PCF × (1 + FPM) × SDV × GPV × SV × RF                       │   ║
║  │                                                                              │   ║
║  │   بشرط:                                                                      │   ║
║  │   • السعر ≥ PCF × PPF (حماية المنتج)                                         │   ║
║  │   • السعر ≤ السعر المرجعي × CPF (حماية المستهلك)                             │   ║
║  │                                                                              │   ║
║  └──────────────────────────────────────────────────────────────────────────────┘   ║
║                                                                                      ║
║  حيث:                                                                               ║
║  • PCF = تكلفة الإنتاج الأساسية (ثابت)                                             ║
║  • FPM = هامش الربح العادل (ثابت)                                                  ║
║  • SDV = متغير العرض والطلب (يومي)                                                 ║
║  • GPV = متغير الأسعار العالمية (لحظي)                                             ║
║  • SV = متغير الموسمية (شهري)                                                      ║
║  • RF = معامل المنطقة (ثابت لكل منطقة)                                             ║
║                                                                                      ║
╚══════════════════════════════════════════════════════════════════════════════════════╝
```

### مثال تطبيقي:

```
مثال: حساب سعر طن حديد تسليح في الرياض

الثوابت:
• PCF = 1,800 ر.س
• FPM = 10% = 1.10
• RF (الرياض) = 1.00

المتغيرات اليومية:
• SDV = 1.05 (طلب زائد طفيف)
• GPV = 1.08 (ارتفاع LME بـ 8%)
• SV = 1.02 (موسم بناء)

الحساب:
السعر = 1,800 × 1.10 × 1.05 × 1.08 × 1.02 × 1.00
السعر = 2,292 ر.س/طن

التحقق:
• الحد الأدنى (PPF): 1,800 × 1.05 = 1,890 ر.س ✅
• الحد الأقصى (CPF): 2,200 × 1.25 = 2,750 ر.س ✅
• السعر 2,292 ر.س ضمن النطاق العادل ✅
```

## 📄 مسودة النظام الأساسي للجمعية

### الباب الأول: أحكام عامة

```
المادة (1): التسمية
═══════════════════
تُؤسس بموجب هذا النظام جمعية أهلية مهنية تُسمى:
"جمعية شيخة للمعادن والسكراب"
Sheikha Metals & Scrap Association (SMSA)

المادة (2): المقر
════════════════
المقر الرئيسي للجمعية: مدينة الرياض، المملكة العربية السعودية
ويجوز إنشاء فروع في أي مكان بقرار من مجلس الإدارة.

المادة (3): الأهداف
═══════════════════
تهدف الجمعية إلى:
أ) تحقيق العدالة السعرية في تجارة المعادن والسكراب
ب) وضع معايير التسعير العادل وفق الشريعة الإسلامية
ج) إصدار مؤشر شيخة للمعادن (SMI) كمرجع سعري
د) تمثيل القطاع لدى الجهات الرسمية
هـ) التوعية والتدريب في مجال المعادن وإعادة التدوير
و) حماية حقوق جميع الأطراف في السوق

المادة (4): الوسائل
═══════════════════
تحقق الجمعية أهدافها بالوسائل التالية:
أ) إنشاء منصة رقمية للتسعير المرجعي
ب) عقد المؤتمرات والندوات والدورات التدريبية
ج) إصدار الدراسات والبحوث والتقارير
د) التنسيق مع الجهات الحكومية والخاصة
هـ) إصدار الشهادات والتراخيص المهنية
و) أي وسيلة مشروعة أخرى

المادة (5): العضوية
═══════════════════
تكون العضوية مفتوحة لكل من:
أ) الشركات والمؤسسات العاملة في قطاع المعادن والسكراب
ب) المصانع والمصاهر
ج) تجار المعادن والسكراب
د) شركات النقل واللوجستيات المتخصصة
هـ) الأفراد المهتمين بالقطاع
و) المنظمات والجمعيات ذات العلاقة
```
