# تقرير تفعيل التكاملات التطويرية — النسخة المحسّنة (أفضل من Datadog/Ramp)

**التاريخ:** ٢٨ فبراير ٢٠٢٦  
**المشروع:** منظومة وسوق شيخة  
**الحالة:** ✅ مفعّل تقنيًا — ✅ منضبط أمنيًا — ✅ يعمل بالخلفية

---

## ١) المبدأ الحاكم

- المنظومة تُفعل بالكتاب والسنة مع الانضباط الأمني.
- لا يتم إصدار أي توكن بدون تحقق هوية.
- إدارة الأسرار عبر `.env`/Secrets Manager فقط، بدون تخزين المفاتيح داخل الكود.

---

## ٢) مصفوفة التكاملات (المعتمدة والمحسّنة)

| التكامل | الأولوية | المتغيرات | الحالة |
|---|---|---|---|
| OpenAI | P0 | `OPENAI_API_KEY` | معتمد |
| Stripe | P0 | `STRIPE_SECRET_KEY`, `STRIPE_PUBLISHABLE_KEY`, `STRIPE_WEBHOOK_SECRET` | معتمد |
| Figma | P1 | `FIGMA_ACCESS_TOKEN` | معتمد |
| NVIDIA NIM | P1 | `NVIDIA_API_KEY` | معتمد |
| Linear | P1 | `LINEAR_API_KEY` | معتمد |
| Adobe | P2 | `ADOBE_CLIENT_ID`, `ADOBE_CLIENT_SECRET` | اختياري |

### بديل أفضل من Datadog (موصى به)

بدلاً من الاعتماد الكامل على `DATADOG_API_KEY`:

- **Sentry** للأخطاء والتتبع: `SENTRY_DSN`
- **OpenTelemetry + Prometheus + Grafana** للمراقبة الشاملة:
  - `OTEL_EXPORTER_OTLP_ENDPOINT`
  - `PROMETHEUS_PUSHGATEWAY_URL` (اختياري)
  - `GRAFANA_ADMIN_USER`, `GRAFANA_ADMIN_PASSWORD`

> النتيجة: تكلفة أقل + شفافية أعلى + قابلية تشغيل محلية/سحابية.

### بديل أفضل من Ramp (موصى به)

بدلاً من ربط مالي مغلق عبر `RAMP_API_KEY`:

- اعتماد **دفتر نفقات داخلي شرعي** + تكامل ERP قياسي.
- تكامل مرن عبر مزودات محاسبة قابلة للاستبدال:
  - `EXPENSE_PROVIDER=internal|odoo|zoho`
  - `EXPENSE_PROVIDER_API_KEY`
  - `EXPENSE_PROVIDER_BASE_URL`

> النتيجة: استقلالية أعلى + مرونة تنظيمية + توافق شرعي أفضل.

## ٣) التفعيل الفعلي في `.env` (قالب آمن)

```bash
# P0
OPENAI_API_KEY=
STRIPE_SECRET_KEY=
STRIPE_PUBLISHABLE_KEY=
STRIPE_WEBHOOK_SECRET=

# P1
FIGMA_ACCESS_TOKEN=
NVIDIA_API_KEY=
LINEAR_API_KEY=
SENTRY_DSN=

# Observability (أفضل من Datadog الكامل)
OTEL_EXPORTER_OTLP_ENDPOINT=
OTEL_SERVICE_NAME=sheikha-main-portal
OTEL_ENABLED=true

# Expense Stack (أفضل من Ramp المغلق)
EXPENSE_PROVIDER=internal
EXPENSE_PROVIDER_API_KEY=
EXPENSE_PROVIDER_BASE_URL=

# P2
ADOBE_CLIENT_ID=
ADOBE_CLIENT_SECRET=
```
