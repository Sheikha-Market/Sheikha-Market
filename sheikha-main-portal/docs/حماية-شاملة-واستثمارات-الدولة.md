## ١. إقرار الاستقلالية

### ١.١ منظومة شيخة — مشروع مستقل

| البند | التفاصيل |
|-------|----------|
| **المالك** | سلمان أحمد بن سلمان الراجح — مالك وحيد |
| **المنشأة** | مؤسسة سلمان احمد بن سلمان الراجح التجارية |
| **السجل** | 2051263653 |
| **الاستقلال** | **شيخة ليست مملوكة لـ ولا تابعة لـ أي جهة حكومية أو استثمارات دولة** |

### ١.٢ كيانات الدولة — نحترمها ولا ننتحل صفتها

استثمارات الدولة تملك شركات مثل:
- **معادن (Ma'aden)** — شركة التعدين السعودية
- **هيوماين (Humain)** — شركة الذكاء الاصطناعي — PIF
- **أرامكو، سابك** — وغيرها

**إقرار صريح:**
- شيخة **لا تدّعي** أي انتماء أو ملكية أو شراكة رسمية مع هذه الكيانات
- شيخة **لا تستخدم** أسماء أو شعارات أو علامات تجارية لهذه الكيانات دون ترخيص
- التكامل مع Humain = استخدام **واجهة API عامة** (Hugging Face) للنموذج ALLaM — مع الاحترام الكامل لشروط الاستخدام والملكية الفكرية لـ Humain

## ٢. حماية الملكية الفكرية

### ٢.١ ملكية شيخة

| النوع | الحماية |
|-------|---------|
| الكود المصدري | ملك سلمان أحمد بن سلمان الراجح — نظام حقوق المؤلف السعودي |
| العلامة التجارية | شيخة / Sheikha — مسجلة ومحمية |
| الأسرار التجارية | خوارزميات التسعير، نماذج AI الداخلية، قواعد المنطق |
| قواعد البيانات | كل البيانات المجمّعة = حق حصري |

### ٢.٢ تكاملات الطرف الثالث — احترام الملكية

| التكامل | المالك | الاستخدام |
|---------|--------|-----------|
| Humain ALLaM | Humain (PIF) | عبر Hugging Face Inference API — ضمن شروط HF |
| OpenAI GPT | OpenAI | ضمن شروط الاستخدام و API |
| Claude | Anthropic | ضمن شروط الاستخدام و API |
| Ollama | مجتمع مفتوح | نماذج مفتوحة المصدر |

**قاعدة:** لا نسخ، لا تعديل للنماذج، لا انتحال — استخدام واجهات رسمية فقط.

## ٣. الأمن

### ٣.١ المفاتيح والأسرار

- كل المفاتيح في `.env` — **لا تُخزَّن في الكود**
- `.env` في `.gitignore` — **لا يُرفع إلى المستودع**
- التحقق من المدخلات — **منع الحقن والتلاعب**

### ٣.٢ الحماية التقنية

- Rate limiting — حماية من الإغراق
- Helmet — ترويسات أمان
- JWT للمصادقة
- لا كشف للمفاتيح في السجلات أو الاستجابات

### ٣.٣ لا ضرر ولا ضرار

- كل عملية تخضع لمبدأ «لا ضرر ولا ضرار»
- فحص شرعي للمدخلات قبل المعالجة
- رفض أي محتوى يمس كيانات الدولة أو ينتحل صفتها

## ٤. التزامات التطوير

1. **عدم الادعاء** بأي علاقة رسمية مع معادن، هيوماين، أرامكو، أو أي كيان حكومي دون وثيقة رسمية
2. **الاحترام الكامل** لشروط استخدام كل API وكل نموذج
3. **الإسناد الصحيح** — عند استخدام نموذج Humain: الإشارة إلى أنه نموذج هيوماين السعودي
4. **حماية البيانات** — عدم تسريب أي بيانات حساسة أو ملكية فكرية لطرف ثالث
