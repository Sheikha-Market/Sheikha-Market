## 1. نظرة عامة

تم تفعيل **الإمكانية الكاملة** لوسوم Meta في منظومة شيخة — تحكم كامل بكيفية ظهور الموقع، أرشفته، وتفاعله مع المتصفحات والمنصات.

---

## 2. القائمة الكاملة — ما تم تفعيله

### 2.1 التحكم في الظهور بمحركات البحث (SEO)

| الوسم | الوظيفة | الحالة |
|-------|----------|--------|
| **description** | نص جذاب يظهر تحت الرابط في جوجل — زيادة نسبة النقر | ✅ مفعّل |
| **robots** | أوامر لمحركات البحث: `index, follow` أو `noindex, nofollow` | ✅ مفعّل |
| **keywords** | كلمات مفتاحية — محركات بحث داخلية ومتخصصة | ✅ مفعّل |

### 2.2 التوافق مع الأجهزة والمتصفحات (Technical)

| الوسم | الوظيفة | الحالة |
|-------|----------|--------|
| **viewport** | استجابة التصميم — `width=device-width, initial-scale=1.0, viewport-fit=cover` | ✅ مفعّل |
| **charset** | ترميز UTF-8 — ضمان عرض النصوص العربية صحيحاً | ✅ مفعّل |
| **Content-Type** | تحديد نوع المحتوى وترميزه | ✅ مفعّل |

### 2.3 التحكم في الانتشار الاجتماعي (Social Media)

| الوسم | المنصة | الحالة |
|-------|--------|--------|
| **og:type** | نوع المحتوى (website) | ✅ مفعّل |
| **og:title** | العنوان عند المشاركة | ✅ مفعّل |
| **og:description** | الوصف | ✅ مفعّل |
| **og:image** | الصورة — فيسبوك، واتساب، لينكدإن | ✅ مفعّل |
| **og:url** | الرابط الكامل | ✅ مفعّل |
| **og:site_name** | اسم الموقع | ✅ مفعّل |
| **og:locale** | اللغة (ar_SA) | ✅ مفعّل |
| **twitter:card** | شكل البطاقة (summary_large_image) | ✅ مفعّل |
| **twitter:site** | @sheikha_market | ✅ مفعّل |
| **twitter:image** | صورة البطاقة لـ X | ✅ مفعّل |

### 2.4 خصائص أمنية وتعريفية

| الوسم/الترويسة | الوظيفة | الحالة |
|----------------|----------|--------|
| **Content-Security-Policy** | حماية من هجمات XSS — تحديد مصادر المحتوى الموثوقة | ✅ عبر Helmet |
| **author** | توثيق ملكية المحتوى | ✅ مفعّل |
| **copyright** | الحقوق الفكرية | ✅ مفعّل |

## 3. الملفات والمسارات

| الملف | الغرض |
|-------|--------|
| `data/sheikha-meta-tags-config.json` | تكوين وسوم Meta لكل الصفحات |
| `public/robots.txt` | أوامر لمحركات البحث — Allow/Disallow |
| `public/sitemap.xml` | خريطة الموقع لمحركات البحث |
| `/api/meta/tags-config` | API لجلب تكوين الوسوم |

---

## 4. الصفحات المحدّثة

- **index.html** — وسوم كاملة (SEO, OG, Twitter, author, copyright)
- **سوق-شيخة.html** — وسوم كاملة
- **robots.txt** — Disallow للوحات التحكم و /api/
- **sitemap.xml** — الصفحات العامة المفهرسة

---

## 5. توصية: صورة og:image

لأفضل ظهور على فيسبوك وواتساب، يُفضّل إنشاء صورة **1200×630 بكسل** (PNG أو JPG) وحفظها في:
```
public/assets/og-image.png
```

ثم تحديث `og:image` في التكوين ليشير إلى:
```
https://sheikha.top/assets/og-image.png
```

حالياً يُستخدم `/icons/icon.svg` كبديل.
