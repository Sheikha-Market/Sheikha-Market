## 4. السوق (Market)

### 4.1 مبادئ سوق المدينة المنورة

- لا ربا — لا غرر — لا غش — لا احتكار — لا نجش
- البيع عن تراضٍ — صدق القول — دقة الوزن
- مؤشر شيخة للمعرفة فقط (ليس تسعير إلزامي)

### 4.2 المكونات

- **سوق المعادن والسكراب** — HS codes، فلاتر، RFQ
- **لوحات التحكم** — أدمن، مستخدم، شركة
- **المجتمع** — منتدى، مختبرات ابتكار
- **الشريعة** — مرجع شرعي، حاسبة زكاة

## 6. جاهزية البنية للتطوير المستمر وفتح أسواق أخرى

### 6.1 مبادئ الجاهزية

| المبدأ | التطبيق | الدليل |
|--------|---------|--------|
| **قابلية التوسع** | خدمات مستقلة، APIs موحدة | وأعدوا لهم ما استطعتم من قوة |
| **عدم التكرار** | DRY — منطق واحد | لا يحب المسرفين |
| **المفتوح-المغلق** | مفتوح للتوسع، مغلق للتعديل | أحسنوا |
| **التوثيق** | فاكتبوه — كل قرار مسجل | البقرة:282 |

### 6.2 فتح أسواق أخرى مستقبلاً

- **معمارية موحدة** — نفس APIs، نفس الامتثال
- **تكاملات قابلة للإضافة** — connectors مستقلة
- **سياسات مرنة** — Policy-as-Code للامتثال
- **بيانات منظمة** — HS codes، مناطق، عملات
