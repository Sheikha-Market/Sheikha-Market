### Balanced Scorecard — 4 منظورات

| المنظور | الوزن | المؤشرات |
|---------|-------|----------|
| الشرعي | 30% | امتثال 100% + مخالفات 0 + تدقيق محتوى 100% |
| التقني | 25% | تشغيل 99.9% + أخطاء 0 + اختراقات 0 + استجابة ≤500ms |
| التجاري | 25% | مستخدمون + شركات + رضا + استمرارية |
| النمو | 20% | معرفة 127+ + محركات 12 + تحسينات + تجارب |
