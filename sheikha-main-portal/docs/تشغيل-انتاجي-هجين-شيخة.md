# تشغيل شيخة الإنتاجي الهجين (Cloud + Local)

## الهدف
- تقوية شيخة عبر تكامل: `Claude + OpenAI + Ollama + LocalMind`
- تشغيل خلفي مستمر
- حوكمة شرعية: **لا ضرر ولا ضرار**

## إعدادات `.env` المقترحة لـ VPS (16GB RAM)
```env
NODE_ENV=production
PORT=8080
HOST=127.0.0.1

# AI Cloud
OPENAI_API_KEY=...
AI_LLM_MODEL=gpt-4o-mini
ANTHROPIC_API_KEY=...
ANTHROPIC_MODEL=claude-opus-4-6-20260205

# Ollama Local
OLLAMA_ENABLED=true
OLLAMA_HOST=http://127.0.0.1:11434
OLLAMA_PROFILE=vps16
OLLAMA_AUTO_UPDATE=true
OLLAMA_UPDATE_INTERVAL_HOURS=24
OLLAMA_ALLOW_PULL=false
OLLAMA_DEFAULT_MODEL=qwen2.5:14b
OLLAMA_STRONG_MODEL=qwen2.5:14b
OLLAMA_LIGHT_MODEL=phi3:mini
OLLAMA_TIMEOUT_MS=90000
```

## مزامنة نماذج Ollama
```bash
cd ~/Projects/sheikha/sheikha-main-portal
npm run ops:ollama:plan
OLLAMA_ALLOW_PULL=true npm run ops:ollama:apply
```

## تشغيل الخدمة بالخلفية (بدون PM2)
> إذا `pm2` غير مثبت، استخدم systemd مباشرة.

### 1) ملف خدمة
أنشئ الملف:
`/etc/systemd/system/sheikha-main.service`

بالمحتوى:
```ini
[Unit]
Description=Sheikha Main Portal
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/root/Projects/sheikha/sheikha-main-portal
ExecStart=/usr/bin/npm run start
Restart=always
RestartSec=5
Environment=NODE_ENV=production

[Install]
WantedBy=multi-user.target
```

### 2) تفعيل الخدمة
```bash
sudo systemctl daemon-reload
sudo systemctl enable --now sheikha-main
sudo systemctl status sheikha-main
```

## نقاط API المهمة
- AI Core status: `/api/ai-core/status`
- Model Router plan: `/api/ai-core/model-router/plan`
- Model Router status: `/api/ai-core/model-router/status`
- Ollama recommendation: `/api/ai-core/ollama/recommendation`
- Akhira status: `/api/akhira/status`
- Akhira signs: `/api/akhira/signs?level=major`
- Akhira important: `/api/akhira/important`
