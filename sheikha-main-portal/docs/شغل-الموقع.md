# تشغيل موقع سوق شيخه محلياً

## ⚠️ إذا ظهر عندك `cd: no such file or directory` أو `Missing script: "start"`

معناه أنك نفّذت الأوامر من **مجلدك الشخصي** (`~`) وليس من مجلد المشروع. استخدم **المسار الكامل** في الأمر الأول (انظر أدناه)، أو افتح مجلد `sheikha-main-portal` في Finder واسحبه إلى الطرفية.

---

## المسار الصحيح للمشروع

المشروع داخل **OneDrive**. من الطرفية (Terminal) استخدم المسار الكامل:

```bash
cd "/Users/salmanalrajeh/Library/CloudStorage/OneDrive-Personal/Sheikha/sheikha-main-portal"
npm install
npm start
```

أو من داخل مجلد **Sheikha** (إذا كنت فيه أصلاً):

```bash
cd sheikha-main-portal
npm install
npm start
```

---

## خطوة بخطوة

1. **افتح Terminal** (أو الطرفية في Cursor).

2. **انتقل لمجلد البوابة:**
   ```bash
   cd "/Users/salmanalrajeh/Library/CloudStorage/OneDrive-Personal/Sheikha/sheikha-main-portal"
   ```

3. **ثبّت التبعيات (مرة واحدة):**
   ```bash
   npm install
   ```

4. **شغّل الخادم:**
   ```bash
   npm start
   ```

5. **افتح المتصفح على:** [http://localhost:8080](http://localhost:8080)

---

## إذا ظهر: `cd: no such file or directory`

- تأكد أنك تكتب المسار كما هو (بما فيه علامات الاقتباس).
- أو من **Finder**: اذهب إلى مجلد المشروع `Sheikha` ثم `sheikha-main-portal`، واسحب المجلد إلى نافذة Terminal ثم اضغط Enter — سيُكتب المسار تلقائياً.

---

## المتطلبات

- **Node.js** (إصدار 14 أو أحدث) — [تحميل Node.js](https://nodejs.org)

---

## إيقاف الخادم

في الطرفية اضغط **Ctrl+C**.

---

## استكشاف الأخطاء

### خطأ: `Missing script: "start"`

معناه أنك لست داخل مجلد `sheikha-main-portal`. نفّذ أولاً:

```bash
cd "/Users/salmanalrajeh/Library/CloudStorage/OneDrive-Personal/Sheikha/sheikha-main-portal"
```

ثم `npm start`.

### خطأ: `EADDRINUSE: address already in use :::8080`

الخادم يعمل أصلاً. افتح [http://localhost:8080](http://localhost:8080) أو أوقف العملية التي تستخدم المنفذ 8080 ثم أعد `npm start`.
