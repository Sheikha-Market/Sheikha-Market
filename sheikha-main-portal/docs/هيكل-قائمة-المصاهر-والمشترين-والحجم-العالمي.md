# هيكل قائمة المصاهر والمشترين والكميات والحجم العالمي

## الغرض

قائمة مرجعية قابلة للتحديث: **المصاهر** و**المشترين** (مصانع، تجار، مصدّرون) مع **الكميات** و**الحجم العالمي** للمشتري المحوّل للسكراب إلى منتج مفيد ونظيف.

## 4. مشترين غير المصاهر — أنواع الحلول

| نوع المشتري | الحل/الاستخدام | أمثلة |
|-------------|-----------------|--------|
| **مصهر** | صهر → مادة خام / منتج وسيط | حديد، نحاس، ألومنيوم |
| **مصنع تصنيع** | شراء معادن/منتجات وسيطة → تصنيع | قطع، منتجات نهائية |
| **تاجر معادن** | شراء سكراب أو معادن → إعادة بيع | تجار جملة، مصدّرون |
| **مصدّر** | تجميع وتصدير | تصدير سكراب أو منتجات |
| **جهة بحث/استثمار** | تحليل بيانات (بدون أسماء أطراف) | تقارير، استثمار |

---

## 5. كيفية تحديث القائمة

1. **مصادر رسمية:** هيئات صناعية، وزارات، هيئات إحصاء.
2. **اتحادات القطاع:** BIR، ISRI، اتحادات إقليمية.
3. **تقارير قطاع:** أبحاث السوق، تقارير شركات.
4. **المنصة:** عند تسجيل مصاهر/مشترين كتجار في سوق شيخه — يمكن (بموافقة) إدراج إحصائيات مجمّعة دون أسماء في التقارير العامة.
