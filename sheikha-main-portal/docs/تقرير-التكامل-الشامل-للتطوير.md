## ٢. تفاصيل كل تكامل

### ٢.١ OpenAI — مفعّل ✅

| البند | التفاصيل |
|-------|----------|
| **الاستخدام** | محادثة ذكية، تحليل أسعار، توصيات، بحث |
| **المتغير** | `OPENAI_API_KEY` في `.env` |
| **API** | `POST /api/ai/chat`، `/api/ai/pricing`، `/api/ai/analytics` |
| **النموذج** | GPT-5.2 (قابل للتغيير عبر `AI_LLM_MODEL`) |

### ٢.٢ Stripe — مدفوعات عالمية

| البند | التفاصيل |
|-------|----------|
| **الاستخدام** | مدفوعات العملاء، اشتراكات، Connect للمنصة |
| **المتغيرات** | `STRIPE_SECRET_KEY`، `STRIPE_PUBLISHABLE_KEY`، `STRIPE_WEBHOOK_SECRET` |
| **التوافق الشرعي** | دعم مرابحة/مضاربة عبر وصف المعاملات |
| **الخطوة** | إضافة `stripe` في `sheikha-payments-engine` |

### ٢.٣ Figma — تصميم تعاوني

| البند | التفاصيل |
|-------|----------|
| **الاستخدام** | استيراد تصميمات، تصدير CSS، مكونات واجهة |
| **المتغير** | `FIGMA_ACCESS_TOKEN` |
| **API** | `https://api.figma.com/v1` |
| **الفائدة** | تصميم → كود أسرع، توحيد الواجهات |

### ٢.٤ NVIDIA — AI وبنية GPU

| البند | التفاصيل |
|-------|----------|
| **الاستخدام** | NVIDIA NIM، NeMo، استدلال محلي/سحابي |
| **المتغير** | `NVIDIA_API_KEY` |
| **الفائدة** | نماذج عربية محلية، تقليل الاعتماد على APIs خارجية |
| **الربط** | Humain (PIF) يستخدم شراكات AWS/NVIDIA/AMD |

### ٢.٥ Datadog — مراقبة وأداء

| البند | التفاصيل |
|-------|----------|
| **الاستخدام** | APM، Logs، Metrics، Synthetics، تنبيهات |
| **المتغيرات** | `DATADOG_API_KEY`، `DATADOG_APP_KEY` |
| **الفائدة** | مراقبة الخادم، تتبع الأخطاء، قياس الأداء |

### ٢.٦ Linear — إدارة مشاريع

| البند | التفاصيل |
|-------|----------|
| **الاستخدام** | مهام، سباقات، تكامل GitHub/GitLab |
| **المتغير** | `LINEAR_API_KEY` |
| **API** | GraphQL `https://api.linear.app/graphql` |
| **الفائدة** | تتبع التطوير، ربط المهام بالكود |

### ٢.٧ Adobe — تصميم وأصول

| البند | التفاصيل |
|-------|----------|
| **الاستخدام** | Adobe XD، Creative Cloud، Experience Platform |
| **المتغيرات** | `ADOBE_CLIENT_ID`، `ADOBE_CLIENT_SECRET` |
| **الفائدة** | أصول مرئية، تجربة مستخدم متقدمة |

### ٢.٨ Ramp — إدارة مصروفات

| البند | التفاصيل |
|-------|----------|
| **الاستخدام** | Spend Management، AI Reporting، Bill Pay |
| **المتغير** | `RAMP_API_KEY` |
| **الفائدة** | ترميز تلقائي، تحكم في الفواتير، تقارير مصروفات |

## ٣. إضافة المتغيرات إلى `.env`

```bash
# ═══ تكاملات التطوير ═══
# OpenAI (مفعّل)
OPENAI_API_KEY=sk-...

# Stripe
STRIPE_SECRET_KEY=sk_...
STRIPE_PUBLISHABLE_KEY=pk_...
STRIPE_WEBHOOK_SECRET=whsec_...

# Figma
FIGMA_ACCESS_TOKEN=...

# NVIDIA
NVIDIA_API_KEY=...

# Datadog
DATADOG_API_KEY=...
DATADOG_APP_KEY=...

# Linear
LINEAR_API_KEY=...

# Adobe
ADOBE_CLIENT_ID=...
ADOBE_CLIENT_SECRET=...

# Ramp
RAMP_API_KEY=...
```
