# نظام إدارة المحتوى والهوية — منظومة وسوق شيخه

## النطاق والهدف

- **نظام إدارة المحتوى:** إدارة نصوص الهوية، الألوان، والعبارات الرئيسية من مكان واحد (هذه الوثيقة + ملف الهوية إن وُجد) — بحيث يمكن تغيير الهوية والألوان بالكامل دون تشتت.
- **ترتيب الظهور:** ١ الهوية الإسلامية أولاً · ٢ هوية التاجر المسلم الأمين الصادق · ٣ تخصص السوق والمنظومة.
- **منظومة وسوق شيخه:** سوق المعادن والسكراب + لوحة تحكم + تكامل + ذكاء — منظومة واحدة؛ التخصص: معادن · سكراب · إعادة تدوير · سلاسل توريد.

---

## 1. ترتيب المحتوى (هوية ذكية بدون تصنّع)

| المحتوى | أين يظهر |
|---------|----------|
| **منظومة وسوق شيخه — تجارة السكراب والمعادن** | الشريط العلوي، البطل، الفوتر، كل صفحة — الهوية الأساسية. |
| **من المصدر إلى المصهر · معادن · سكراب · إعادة تدوير · سلاسل توريد** | البطل، الفوتر — التخصص. |
| **أمانة وصدق في التعامل — ما تلتزم به المنظومة وتُتيحه للتاجر** | البطل، الفوتر — تحقيق فعلي بالآليات، لا شعارات. |
