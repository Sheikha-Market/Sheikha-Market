# 📈 المصدر الأول: عمولات المبيعات (35% من الإيرادات)

## 1.1 عمولة سوق شيخة

### آلية العمولة

```
┌─────────────────────────────────────────────────────────────────┐
│                     رحلة عمولة البيع                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   البائع يعرض        المشتري يشتري        إتمام الصفقة          │
│   منتج بـ 100,000 ر.س    ←────────→     المبلغ يُحجز           │
│                                                                 │
│                           ↓                                     │
│                                                                 │
│   تأكيد الاستلام        خصم العمولة        تحويل للبائع          │
│   من المشتري     →     (3% = 3,000 ر.س)  →  97,000 ر.س          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### جدول العمولات حسب الباقة

| باقة التاجر | نسبة العمولة | الحد الأدنى للعمولة |
|-------------|-------------|---------------------|
| الأساسية (مجانية) | 5% | 50 ر.س |
| المتقدمة (500 ر.س/شهر) | 3% | 30 ر.س |
| الاحترافية (1,500 ر.س/شهر) | 2% | 20 ر.س |
| المؤسسية (تفاوض) | 1-1.5% | تفاوض |

### جدول العمولات حسب نوع المعدن

| نوع المعدن | عمولة إضافية | السبب |
|------------|-------------|-------|
| معادن أساسية | 0% | هامش ربح منخفض |
| معادن ثمينة | +0.5% | قيمة عالية، مخاطر أقل |
| معادن نادرة | +0.5% | سوق متخصص |
| سكراب | 0% | حجم كبير، هامش منخفض |

### مثال حسابي للعمولات

```
سيناريو: تاجر بالباقة المتقدمة يبيع:
├── 100 طن حديد × 3,000 ر.س = 300,000 ر.س
│   └── عمولة: 300,000 × 3% = 9,000 ر.س
│
├── 50 كجم نحاس × 50 ر.س = 2,500 ر.س
│   └── عمولة: 2,500 × 3% = 75 ر.س
│
├── 1 كجم ذهب × 250,000 ر.س = 250,000 ر.س
│   └── عمولة: 250,000 × 3.5% = 8,750 ر.س
│
└── إجمالي العمولات = 17,825 ر.س
```

## 1.2 عمولة الوساطة

### للصفقات الكبيرة والعقود

| حجم الصفقة | نسبة الوساطة |
|------------|-------------|

## المبادئ الشرعية المطبقة

### 1. حلية مصادر الربح

```
✅ مصادر حلال:
├── عمولة على وساطة حقيقية
├── رسوم خدمات فعلية
├── اشتراكات مقابل مميزات
├── إيجار مستودعات
└── أرباح تجارة مشروعة

❌ مصادر محرمة (ممنوعة):
├── فوائد ربوية
├── غرامات تأخير ربوية
├── تجارة محرمات
└── غرر وجهالة
```

### 2. العدالة في التسعير

> **قال ﷺ: "دعوا الناس يرزق الله بعضهم من بعض"**

- أسعار تنافسية ومعلنة
- لا احتكار أو تلاعب
- حرية المفاوضة
- شفافية كاملة

### 3. توزيع القيمة

```
القيمة المضافة تُوزع بعدالة:
├── البائع: يحصل على سعر عادل
├── المشتري: يحصل على جودة بسعر مناسب
├── الناقل: يحصل على أجر عمله
├── المنصة: تحصل على مقابل خدماتها
└── المجتمع: فرص عمل، تطوير القطاع
```
