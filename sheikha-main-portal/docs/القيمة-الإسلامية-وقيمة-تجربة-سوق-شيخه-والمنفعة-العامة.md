# القيمة الإسلامية وقيمة تجربة سوق شيخه — والمنفعة العامة

## النطاق

- **قيمة إسلامية** — وصف قيم سوق شيخه في ضوء الإسلام (حلال، أمانة، شفافية، بلا ضرر ولا ضرار).
- **قيمة التجربة** — وصف قيمة تجربة السوق بصفة **أدمن**، **تاجر**، وقيمتها بالنسبة لل**عميل** وتجربته.
- **قيمة للإسلام والسعودية والعالم** — وصف قيمتها للإسلام، المملكة العربية السعودية (دولتي)، والعالم أجمعين.
- **قيمة للبيئة والتجارة وسلاسل الإمداد والتوريد** — وما يمكن أن يكون **مكسباً ومنفعة عامة**.

---

# الجزء الأول — القيمة الإسلامية

## 1.1 المبادئ الإسلامية في سوق شيخه

| المبدأ | التطبيق |
|--------|---------|
| **الحلال** | تجارة حلال؛ محتوى حلال (لا صور ذوات أرواح)؛ مدفوعات وعقود بضوابط الشريعة؛ زخارف هندسية ونخيل ونجوم فقط. |
| **الأمانة والشفافية** | تتبع من المصدر إلى المنتج؛ أسعار معلنة؛ مصدر الأسعار واضح؛ لا عمولات مخفية — أمانة في التعامل وشفافية في العرض. |
| **بلا ضرر ولا ضرار** | لا نهدف من أي منتج أو خدمة إلى إلحاق ضرر بالبيئة أو بالصحة أو بالبشر أو بالمخلوقات؛ التزام صريح بعدم الإضرار. |
| **الصحة وقاية** | من هدي النبي ﷺ (لا عدوى ولا طيرة، والأخذ بالأسباب) — المحافظة على الصحة بالوقاية؛ إعادة التدوير تساهم في وقاية البيئة والصحة. |
| **حب العلامة وأمان** | العلامة مرتبطة بالأمان والجودة والمحافظة — لا استغلال ولا خداع؛ ثقة وأمان للجميع. |

## 1.2 القيمة الإسلامية — ملخص

- سوق شيخه تُبنى على **حلال، أمانة، شفافية، بلا ضرر ولا ضرار، وقاية، وأمان** — قيم إسلامية تُترجم إلى تجربة وتجارة وبيئة يحسّن فيها المكسب والمنفعة العامة دون إضرار.

# الجزء الثاني — قيمة التجربة (أدمن، تاجر، عميل)

## 2.1 قيمة التجربة بصفة أدمن

| البند | الوصف |
|-------|--------|
| **نظرة شاملة** | لوحة تحكم ونظم إدارية موحدة — إحصائيات (متاجر، عروض، عمليات)، إدارة المستخدمين والتجار والعروض، التقارير، الإعدادات. |
| **مؤشرات الأداء** | مؤشرات تسويق (زوار، مشاهدات صفحات، تحويلات) ومؤشرات أداء (عمليات، أرشفة، سحابة) — كلها في داشبورد واحد. |
| **التحكم والجودة** | إدارة المحتوى، الامتثال للحلال واللوائح، مراقبة سلاسل الإمداد والعمليات — لضمان جودة السوق والمنفعة العامة. |
| **القيمة للأدمن** | أداة واحدة للرقابة والإدارة والتخطيط — أفضل من تشتت الأنظمة بدون رؤية موحدة. |

## 2.2 قيمة التجربة بصفة تاجر

| البند | الوصف |
|-------|--------|
| **تسجيل وعرض في خطوة واحدة** | تسجيل متجر بتخصص (مصدر، جامع، ناقل، مصهر، معادن، شاحنات) ثم عرض بضائع — مسار واضح وقصير؛ أفضل من أي سوق للمتاجر. |
| **ربط بمرحلة السلسلة ونوع العرض** | كل عرض مرتبط بمرحلة سلسلة التوريد ونوع السكراب (تجميع، صهر، نقل، شاحنات) — يظهر في المكان الصحيح في السوق. |
| **داشبورد تاجر** | متجري، منتجاتي، إضافة منتج، إشعارات، تسويق، إعدادات — واجهة واحدة لإدارة المتجر والعروض. |
| **القيمة للتاجر** | وصول أسرع للمشترين، شفافية أسعار، سلسلة توريد واضحة — مكسب تجاري مع أمان وثقة. |

## 2.3 قيمة التجربة بالنسبة للعميل وتجربته

| البند | الوصف |
|-------|--------|
| **وضوح المسار** | ثلاث خطوات: سجّل متجرك، اعرض بضاعتك، تواصل وبيع — العميل (زائر أو مشتري) يفهم فوراً كيف يدخل السوق ويجد ما يريد. |
| **سوق متخصص 100%** | معادن وسكراب فقط — عميل يبحث عن سكراب أو معادن يجد عروضاً مصنّفة حسب المرحلة والنوع دون تشتت. |
| **أسعار مدمجة مع العروض** | أسعار مرجعية (عالمية، سعودية) في نفس الصفحة مع عروض المتاجر — العميل يقارن ويقرر بثقة. |
| **تجربة آمنة وحلال** | محتوى حلال؛ شفافية؛ بلا ضرر ولا ضرار — تجربة مريحة وآمنة للعميل. |
| **القيمة للعميل** | إيجاد عرض مناسب بسرعة، ثقة في المصدر والسعر، تواصل واحد (واتساب أو المنصة) — منفعة واضحة وتجربة أفضل من أسواق عامة غير متخصصة. |

# الجزء الثالث — القيمة للإسلام والسعودية والعالم

## 3.1 القيمة للإسلام

| البند | الوصف |
|-------|--------|
| **تجارة حلال نموذجية** | منصة تجارة تُظهر أن الحلال والشفافية والأمانة ممكنة في السوق الرقمي — نموذج يُحتذى. |
| **قيم إسلامية مطبّقة** | بلا ضرر ولا ضرار، وقاية، أمان — قيم تُترجم إلى سياسات وتجربة فعلية. |
| **إعادة تدوير ومحافظة على البيئة** | إعادة التدوير من منظور «لا إفساد في الأرض» ومحافظة على الموارد — انسجام مع مقاصد الشريعة. |

## 3.2 القيمة للمملكة العربية السعودية (دولتي)

| البند | الوصف |
|-------|--------|
| **سوق وطنية متخصصة** | سوق معادن وسكراب وإعادة تدوير تخدم الاقتصاد المحلي والتصنيع والاستدامة — دعم رؤية 2030 في الاقتصاد والبيئة. |
| **سلاسل إمداد محلية وإقليمية** | ربط المصادر والمصاهر والتجار داخل المملكة والخليج — تقوية سلاسل التوريد الوطنية. |
| **توظيف وريادة أعمال** | تسهيل دخول التجار والمتاجر الصغيرة — فرص عمل وريادة أعمال في قطاع المعادن والسكراب. |
| **امتثال ولوائح** | تكامل مع اللوائح والإجراءات (نفاذ، مدن ذكية) — انسجام مع البيئة التنظيمية السعودية. |

## 3.3 القيمة للعالم أجمعين

| البند | الوصف |
|-------|--------|
| **سوق عالمية متخصصة** | معادن وسكراب وإعادة تدوير — حاجة عالمية؛ المنصة قابلة للتوسع لدول أخرى. |
| **شفافية وسلاسل توريد** | تتبع من المصدر إلى المنتج — معيار يمكن أن يُعمّم لتحسين سلاسل التوريد عالمياً. |
| **بيئة واستدامة** | إعادة التدوير والمحافظة على البيئة — منفعة عالمية (تقليل انبعاثات، حماية موارد). |
| **تجربة أفضل من أي سوق للمتاجر** | آلية تسجيل وعرض متخصصة — نموذج قابل للاقتداء في أسواق أخرى. |

# الجزء الرابع — القيمة للبيئة والتجارة وسلاسل الإمداد — المكسب والمنفعة العامة

## 4.1 القيمة للبيئة

| البند | الوصف |
|-------|--------|
| **إعادة التدوير** | كل طن معاد تدويره = تقليل استخراج خام، تقليل انبعاثات، تقليل مخلفات — محافظة على البيئة. |
| **بلا ضرر ولا ضرار** | التزام صريح بعدم الإضرار بالبيئة؛ شهادات بيئية وقياس (نسبة وكمية) — منفعة عامة بيئية. |
| **مكسب عام** | بيئة أنظف، صحة أفضل، موارد محفوظة — منفعة للمجتمع والكائنات. |

## 4.2 القيمة للتجارة

| البند | الوصف |
|-------|--------|
| **تسهيل البيع والشراء** | ربط البائع بالمشتري في سوق واحد متخصص — تقليل تكلفة البحث والوساطة. |
| **شفافية الأسعار** | أسعار مرجعية وعروض واضحة — تجارة عادلة وثقة. |
| **مكسب للتاجر والمشتري** | تاجر يصل لمشترين أسرع؛ مشتري يجد عرضاً مناسباً — منفعة متبادلة. |

## 4.3 القيمة لسلاسل الإمداد والتوريد

| البند | الوصف |
|-------|--------|
| **مراحل واضحة** | مصدر → تجميع → نقل → مصهر → تصنيع → طرف نهائي — كل طرف يعرف مكانه في السلسلة. |
| **تتبع وجودة** | قياس جودة السلاسل ومؤشرات بيئية — تحسين سلاسل الإمداد والتوريد. |
| **مكسب عام** | سلاسل أقوى، أقل هدراً، أكثر شفافية — منفعة للاقتصاد والصناعة. |

## 4.4 المكسب والمنفعة العامة — ملخص

| المجال | المكسب / المنفعة العامة |
|--------|---------------------------|
| **الإسلام** | نموذج تجارة حلال، قيم مطبّقة، إعادة تدوير ومحافظة. |
| **السعودية** | سوق وطنية، سلاسل محلية، توظيف، امتثال. |
| **العالم** | سوق متخصصة قابلة للتوسع، شفافية، بيئة واستدامة. |
| **البيئة** | إعادة تدوير، تقليل ضرر، محافظة على الموارد. |
| **التجارة** | تسهيل بيع وشراء، شفافية، مكسب للتاجر والمشتري. |
| **سلاسل الإمداد** | مراحل واضحة، تتبع وجودة، سلاسل أقوى. |
| **الأدمن** | نظرة شاملة، مؤشرات أداء، تحكم وجودة. |
| **التاجر** | مسار قصير، وصول أسرع، داشبورد موحّد. |
| **العميل** | وضوح، تخصص، أسعار، تجربة آمنة وحلال. |
