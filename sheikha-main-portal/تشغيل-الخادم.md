# تشغيل خادم سوق شيخه

## المشكلة: `npm error Missing script: "start"`

يحدث هذا عندما تُشغّل `npm start` من **مجلد خاطئ** (مثل مجلد المنزل `~`). السكربت `start` موجود داخل مشروع **sheikha-main-portal** فقط.

## الحل — تشغيل من مجلد المشروع

1. **انتقل إلى مجلد المشروع:**
   ```bash
   cd "/Users/salmanalrajeh/Library/CloudStorage/OneDrive-Personal/Sheikha/sheikha-main-portal"
   ```

2. **ثم شغّل الخادم:**
   ```bash
   nvm use 20
   npm start
   ```

   أو مباشرة:
   ```bash
   node server.js
   ```

3. **افتح المتصفح:**  
   [http://localhost:8080](http://localhost:8080)

## سكربت واحد (نسخ ولصق)

```bash
cd "/Users/salmanalrajeh/Library/CloudStorage/OneDrive-Personal/Sheikha/sheikha-main-portal" && nvm use 20 && npm start
```

## إذا ظهر خطأ libc++ (Symbol not found)

استخدم Node 20 مباشرة:

```bash
cd "/Users/salmanalrajeh/Library/CloudStorage/OneDrive-Personal/Sheikha/sheikha-main-portal"
/Users/salmanalrajeh/.nvm/versions/node/v20.20.0/bin/node server.js
```

## إذا ظهر "address already in use :::8080"

معناه أن الخادم **يعمل بالفعل** على المنفذ 8080. افتح المتصفح على: [http://localhost:8080](http://localhost:8080)

---

الخادم يعمل على **المنفذ 8080** ما لم تُغيّر `PORT` في البيئة أو في `server.js`.
