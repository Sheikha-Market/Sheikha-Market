## 🚀 الاستخدام السريع

### تحميل النظام الموحد:

```javascript
const Unified = require('./lib/sheikha-unified-agents-system.js');
```

### الحصول على جميع الوكلاء:

```javascript
const agents = Unified.getAllAgents();
console.log(`الإجمالي: ${agents.total} وكيل`);
```

### البحث عن وكيل:

```javascript
const result = Unified.findAgent('agent-cfo');
console.log(result.agent.nameAr); // "المدير المالي التنفيذي"
```

### الحصول على إحصائيات:

```javascript
const stats = Unified.getSystemStatistics();
console.log(stats);
```

### تفعيل وكيل:

```javascript
const active = Unified.activateAgent('agent-training-manager', {
    mode: 'production',
    language: 'ar'
});
```

### اقتراح وكلاء حسب الحاجة:

```javascript
const suggestions = Unified.suggestAgents({
    finance: true,
    security: true,
    health: true
});
```
