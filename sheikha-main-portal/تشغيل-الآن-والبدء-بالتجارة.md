# تشغيل الموقع الآن والبدء بالتجارة

## خطوات التشغيل (انسخ والصق في الطرفية بالترتيب)

### 1) تحميل nvm واستخدام Node 20 (لتجنب خطأ dyld على macOS 12)

```bash
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
nvm use 20
```

إذا ظهر أن Node 20 غير موجود، نفّذ أولاً:
```bash
nvm install 20
nvm use 20
```

### 2) إزالة نسخة Node التي تسبب الخطأ (إن وُجدت)

```bash
rm -rf "$HOME/node_modules/node"
```

### 3) الانتقال لمجلد المشروع وتشغيل الخادم

```bash
cd ~/Library/CloudStorage/OneDrive-Personal/Sheikha/sheikha-main-portal
npm start
```

### 4) فتح الموقع في المتصفح

عند ظهور رسالة مثل: **Server running on port 8080**، افتح في المتصفح:

**http://localhost:8080**

---

## البدء بالتجارة

| الخطوة | ماذا تفعل |
|--------|-----------|
| **1. الرئيسية** | من الصفحة الرئيسية يمكنك استكشاف الأسعار، السوق، وروابط التسجيل وعرض البضائع. |
| **2. تسجيل كتاجر** | اضغط «تسجيل تاجر» أو افتح: **http://localhost:8080/تسجيل-تاجر.html** — أدخل الاسم والجوال والبريد والمنطقة ثم «تسجيل التاجر». |
| **3. عرض بضائع (بيع سكراب)** | اضغط «عرض بضائع» أو افتح: **http://localhost:8080/عرض-بضائع.html** — اختر التاجر أو أدخل رقم جوالك، ثم أدخل اسم البضاعة والكمية والسعر واضغط «إضافة عرض البضاعة». |
| **4. واتساب للبيع/الشراء فوراً** | من الصفحة الرئيسية أو صفحة ريادة الأعمال: اضغط «أريد البيع الآن — واتساب» أو «أريد الشراء الآن — واتساب» للتواصل على **+966 55 494 2904**. |
| **5. ريادة الأعمال** | افتح **http://localhost:8080/ريادة-الأعمال.html** لخطوات الانطلاق والأرباح. |

---

## روابط سريعة بعد التشغيل

- الرئيسية: http://localhost:8080  
- تسجيل تاجر: http://localhost:8080/تسجيل-تاجر.html  
- عرض بضائع: http://localhost:8080/عرض-بضائع.html  
- ريادة الأعمال: http://localhost:8080/ريادة-الأعمال.html  

---

**إيقاف الخادم:** في الطرفية اضغط `Ctrl+C`.
