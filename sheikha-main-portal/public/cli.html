التثبيت

ثلاث طرق لتثبيت Sheikha CLI

npm
curl
brew
# تثبيت Sheikha CLI عبر npm
npm install -g @sheikha/cli
 
# التحقق من التثبيت
sheikha --version
# sheikha-cli/1.4.0 linux-x64 node/v20.11.0
 
# تسجيل الدخول
sheikha login
# ✓ مرحباً! تم تسجيل الدخول كـ trader@example.com
يتطلب Node.js 18+ | متوافق مع Linux، macOS، Windows

الأوامر

دليل الأوامر الكامل مصنفةً حسب المجموعة

📡
sheikha status
فحص حالة المنظومة
sheikha status
عرض حالة جميع خدمات منظومة شيخة
sheikha status --json
إخراج الحالة بصيغة JSON للمعالجة البرمجية
sheikha status --watch
مراقبة الحالة بشكل حي (تحديث كل 5 ثوانٍ)
🏪
sheikha market
أوامر السوق الحي
sheikha market live
عرض أسعار المعادن اللحظية من LME/COMEX/LBMA
sheikha market prices --metal gold
سعر معدن محدد مع التاريخ والتغيير
sheikha market offers --list
قائمة العروض النشطة في السوق
sheikha market order create
إنشاء طلب شراء أو بيع جديد
📋
sheikha contracts
إدارة العقود الرقمية
sheikha contracts inspect <id>
فحص تفاصيل عقد بمعرفه
sheikha contracts list
قائمة عقودك النشطة والمكتملة
sheikha contracts create
إنشاء عقد رقمي جديد (تفاعلي)
sheikha contracts approve <id>
الموافقة على عقد معلق
🔗
sheikha supply
إدارة سلاسل الإمداد
sheikha supply sync
مزامنة بيانات الإمداد مع الموردين
sheikha supply track <order-id>
تتبع شحنة أو أمر توريد
sheikha supply suppliers --list
قائمة الموردين المرتبطين
⚖️
sheikha governance
فحوصات الحوكمة والامتثال
sheikha governance check
تشغيل فحص الامتثال الكامل للميثاق
sheikha governance audit --tail 50
عرض آخر 50 سجل في سجل الأثر
sheikha governance policies
قائمة السياسات التشغيلية النشطة
⚡
sheikha sdk
إدارة مجموعة أدوات التطوير
sheikha sdk list
قائمة حزم SDK المتاحة وإصداراتها
sheikha sdk keys --create
إنشاء مفتاح API جديد
sheikha sdk test --endpoint /api/market
اختبار endpoint محدد
🔮
sheikha mcp
إدارة مركز ربط الأدوات
sheikha mcp health
فحص صحة جميع عقد MCP
sheikha mcp tools --list
قائمة الأدوات المسجلة في MCP
sheikha mcp connect --node market
الاتصال بعقدة محددة
🌐
sheikha git
تكامل GitHub وGitLab
sheikha git status
حالة المستودعات والفروع والتزامن
sheikha git branches
قائمة الفروع وحالة الدمج
sheikha git sync
مزامنة GitHub وGitLab
sheikha git report
توليد تقرير نشاط التطوير اليومي
