التثبيت الآمن الرسمي مع التحقق والتوثيق.

1) التثبيت الطرفي (موصى به)

شغّل الأوامر التالية:

sudo bash scripts/install.sh

قبل التشغيل العام، حدّث ملف .env بالقيم السرية.

2) حزمة Debian (.deb)

ابن الحزمة:

bash scripts/build-deb.sh

ثم ثبّت:

sudo dpkg -i dist/sheikha-main-portal_1.0.0_amd64.deb

تحقق من checksum:

sha256sum -c dist/sheikha-main-portal_1.0.0_amd64.deb.sha256

3) تحقق السلامة قبل الإصدار

bash scripts/validate-release.sh
  • فحص أمني للاعتمادات
  • توليد SBOM
  • توليد checksums
  • التحقق من ملفات الحماية والتوزيع

أمر التثبيت:

curl -fsSL https://www.sheikha.top/scripts/install.sh -o install.sh && sudo bash install.sh

sudo dpkg -i sheikha-main-portal_1.0.0_amd64.deb

تنزيل شيخة — القناة الرسمية

قناة توزيع موثقة وآمنة للعملاء. جميع الحزم الرسمية يجب التحقق منها قبل التثبيت.

1) التثبيت عبر الطرفية (موصى به)

ثبّت مباشرة من الحزمة الرسمية:

sudo bash scripts/install.sh

إزالة التثبيت:

sudo bash scripts/uninstall.sh

2) حزمة Debian (.deb)

ابنِ الحزمة:

npm run release:deb

ثبّت الحزمة:

sudo dpkg -i dist/sheikha-main-portal_1.0.0_amd64.deb

3) التحقق الأمني قبل الإصدار

npm run release:validate

سيتم إنتاج ملفات: sbom.cdx.json و release-checksums.sha256.
