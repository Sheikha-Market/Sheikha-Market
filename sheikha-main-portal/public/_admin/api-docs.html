🖥️ Server APIs

GET /api/server/status حالة الخادم والأداء ▼
Response
{ "success": true, "server": { "status": "online", "uptime": { "ms": 3600000, "formatted": "1h 0m 0s" }, "version": "2.0.0", "nodeVersion": "v20.x.x" }, "performance": { "requestsTotal": 1284, "avgResponseTime": "45ms" }, "memory": { ... } }
GET /api/server/metrics مقاييس الأداء التفصيلية ▼
Response
{ "success": true, "metrics": { "uptime": { "seconds": 3600, "percentage": "99.9%" }, "requests": { "total": 1284, "perHour": 1284 }, "response": { "average": 45, "unit": "ms" }, "memory": { "heapUsedMB": 128, "percentUsed": 68 } } }
GET /api/server/services حالة جميع الخدمات ▼
Response
{ "success": true, "services": [ { "id": "web-server", "status": "online" }, { "id": "ai-service", "status": "online" }, { "id": "websocket", "status": "online" }, ... ] }

❤️ Health Checks

GET /api/health فحص صحة النظام ▼
Response
{ "success": true, "status": "healthy", "checks": { "server": true, "database": true, "memory": true } }
GET /api/ready جاهزية النظام ▼
Response
{ "success": true, "ready": true, "uptime": 3600000 }

🔐 Authentication APIs

Request Body
{ "phone": "05xxxxxxxx", "password": "your-password" }
Response
{ "success": true, "token": "eyJhbGciOiJIUzI1NiIs...", "user": { ... } }
POST /api/auth/register تسجيل جديد ▼
Request Body
{ "name": "اسم المستخدم", "phone": "05xxxxxxxx", "password": "your-password", "storeName": "اسم المتجر", "region": "الرياض" }
GET /api/me بيانات المستخدم الحالي Auth ▼
Headers
Authorization: Bearer {token}
