# دليل السلامة الإدارية الكامل

## 1) الهدف
ضمان وجود نظام إداري آمن وواضح المسؤوليات يمنع التعارض، يقلل المخاطر، ويرفع جودة القرار.

## 2) النطاق
- الإدارة العليا
- إدارة العمليات
- إدارة الجودة والامتثال
- إدارة الموارد البشرية

## 3) السياسات الأساسية
1. اعتماد سياسة سلامة إدارية مكتوبة ومعلنة.
2. تحديد الصلاحيات والمسؤوليات (RACI).
3. منع تضارب المصالح والإفصاح الإلزامي.
4. التوثيق الكامل للقرارات المحورية.
5. مراجعة دورية لنقاط الضعف الإدارية.

## 4) إجراءات التشغيل القياسية (SOP)
### أ) بدء العمل اليومي
- مراجعة لوحة المخاطر.
- تأكيد جاهزية الفرق.
- اعتماد أولويات اليوم.

### ب) اعتماد قرار تشغيلي
- تحديد الأثر المالي والتشغيلي.
- فحص التوافق النظامي والشرعي.
- تسجيل القرار في سجل القرارات.

### ج) إغلاق اليوم
- مراجعة الحوادث/الملاحظات.
- توثيق الانحرافات وخطة المعالجة.
- تحديث مؤشرات الأداء.

## 5) مؤشرات المتابعة
- نسبة الالتزام بالإجراءات
- عدد الانحرافات المفتوحة
- زمن إغلاق الملاحظة
- نسبة القرارات الموثقة

## 6) نموذج تدقيق أسبوعي
- هل تم تحديث سجل المخاطر؟
- هل توجد قرارات غير موثقة؟
- هل تم تنفيذ اجتماعات المراجعة؟
- هل تم رفع التقارير في موعدها؟

## 7) التحسين المستمر
- دورة: خطط → نفّذ → تحقّق → حسّن (PDCA)
- مراجعة شهرية مع خطة تصحيح واضحة.
