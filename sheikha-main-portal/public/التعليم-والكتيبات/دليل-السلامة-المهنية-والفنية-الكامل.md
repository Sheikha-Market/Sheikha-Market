# دليل السلامة المهنية والفنية الكامل

## 1) الهدف
حماية العاملين والمستخدمين والأنظمة من الإصابات والأعطال عبر ممارسات مهنية وفنية منضبطة.

## 2) ضوابط السلامة المهنية
- تدريب إلزامي قبل مباشرة أي مهمة.
- استخدام معدات الوقاية المناسبة للمهمة.
- منع العمل دون تصريح مهني عند الأعمال عالية الخطورة.
- الإبلاغ الفوري عن أي حادث أو شبه حادث.

## 3) ضوابط السلامة الفنية
- فحص قبل التشغيل (Pre-check).
- فحص بعد التشغيل (Post-check).
- صيانة وقائية وفق جدول معتمد.
- خطة نسخ احتياطي واستعادة للنظم الرقمية.

## 4) قائمة التحقق اليومية
1. فحص بيئة العمل.
2. فحص الأدوات والأنظمة.
3. تأكيد صلاحية خطط الطوارئ.
4. توثيق الملاحظات الفنية.
5. التوقيع على نموذج الجاهزية.

## 5) حالات التوقف الإلزامي
- ظهور خطر مباشر على الأشخاص.
- عطل فني قد يسبب ضررًا متسلسلًا.
- نقص شرط أمان إلزامي.

## 6) آلية البلاغات
- القناة: نموذج حادث + إشعار فوري للمسؤول.
- زمن الاستجابة: فوري للحوادث الحرجة.
- التوثيق: السبب، الأثر، الإجراء، الإغلاق.

## 7) تدريب وتوعية
- دورة تعريفية عند الانضمام.
- ورشة أسبوعية قصيرة (15–20 دقيقة).
- اختبار شهري لقياس الاستيعاب.
