# دليل سلامة إدارة المشاريع الكامل

## 1) الهدف
دمج السلامة في كل مرحلة من مراحل المشروع لضمان التنفيذ الآمن ضمن الوقت والتكلفة والجودة.

## 2) المراحل
1. التهيئة
2. التخطيط
3. التنفيذ
4. المراقبة والتحكم
5. الإغلاق

## 3) ضوابط السلامة لكل مرحلة
### التهيئة
- تعريف المخاطر الرئيسية.
- تحديد مسؤول السلامة في المشروع.

### التخطيط
- سجل مخاطر تفصيلي.
- خطة استجابة لكل خطر.
- اعتماد نقاط توقف (Gate Reviews).

### التنفيذ
- متابعة يومية للمخاطر.
- تفعيل تصاريح العمل عند الحاجة.
- منع تخطي الإجراءات.

### المراقبة والتحكم
- قياس الانحرافات.
- إجراءات تصحيحية فورية.
- تقارير أسبوعية للإدارة.

### الإغلاق
- مراجعة الدروس المستفادة.
- توثيق حالات النجاح والإخفاق.
- إغلاق جميع المخاطر المفتوحة.

## 4) مصفوفة تصعيد
- منخفض: فريق المشروع.
- متوسط: مدير المشروع + الجودة.
- مرتفع: الإدارة العليا + لجنة السلامة.

## 5) مؤشرات سلامة المشروع
- عدد الحوادث/شبه الحوادث.
- نسبة المخاطر المغلقة.
- زمن الاستجابة للحالات الحرجة.
- نسبة الالتزام بنقاط التوقف.

# 📐 منظومة القياس والدراسة الشاملة للمشاريع

## 🎯 الهدف
قياس ودراسة كل مشروع (توريد، تصدير، خدمة) بشكل نموذجي شامل مع تفعيل وكلاء ذكاء اصطناعي متخصصين لضمان:
- دراسة جدوى متكاملة
- تحديد الإجراءات والمعدات اللازمة
- التخطيط والتنفيذ الأمثل
- التوصيل الآمن بصدق وأمانة

---

## 🤖 الوكلاء المتخصصون لدراسة المشاريع

### 1. وكيل دراسة الجدوى المالية
**المهام:**
- تحليل التكاليف والعوائد المتوقعة
- حساب فترة الاسترداد ROI
- تحليل الربحية ونقطة التعادل
- تقييم المخاطر المالية

**المخرجات:**
- تقرير الجدوى المالية الشامل
- مؤشرات مالية (NPV, IRR, Payback Period)
- توصية بالموافقة أو الرفض أو التعديل

---

### 2. وكيل دراسة الجدوى الفنية
**المهام:**
- تقييم الجدوى التقنية للمشروع
- تحديد المعدات والتقنيات المطلوبة
- تقييم القدرات الفنية المتاحة
- تحديد الفجوات والاحتياجات

**المخرجات:**
- قائمة المعدات والأجهزة المطلوبة
- المواصفات الفنية التفصيلية
- خطة التدريب الفني
- تقييم المخاطر الفنية

---

### 3. وكيل دراسة السوق والطلب
**المهام:**
- تحليل السوق المستهدف
- تقييم حجم الطلب والعرض
- دراسة المنافسين
- تحليل الأسعار والاتجاهات

**المخرجات:**
- تقرير تحليل السوق
- حجم السوق المتوقع
- استراتيجية التسعير
- الفرص والتهديدات

---

### 4. وكيل التخطيط والجدولة
**المهام:**
- إعداد الجدول الزمني التفصيلي
- تحديد المسار الحرج (Critical Path)
- توزيع الموارد والمهام
- تحديد معالم المشروع (Milestones)

**المخرجات:**
- خطة المشروع التفصيلية (Gantt Chart)
- تحديد المسار الحرج
- جدول الموارد
- خطة إدارة المخاطر

---

### 5. وكيل إدارة المشتريات
**المهام:**
- تحديد احتياجات المشتريات
- البحث عن الموردين وتقييمهم
- إدارة العقود والمناقصات
- ضمان جودة التوريد

**المخرجات:**
- قائمة المشتريات الشاملة
- تقييم الموردين
- عقود التوريد
- جدول التسليم

---

### 6. وكيل الامتثال والمطابقة
**المهام:**
- التحقق من الامتثال للمعايير والقوانين
- مراجعة التراخيص والشهادات
- ضمان المطابقة الشرعية الإسلامية
- التحقق من معايير الجودة

**المخرجات:**
- تقرير الامتثال الكامل
- قائمة التراخيص المطلوبة
- شهادات المطابقة
- توثيق الامتثال الشرعي

---

### 7. وكيل اللوجستيات والشحن
**المهام:**
- تخطيط النقل والشحن
- اختيار وسائل النقل المناسبة
- إدارة المستودعات
- تتبع الشحنات

**المخرجات:**
- خطة اللوجستيات الشاملة
- اختيار شركات الشحن
- جدول الشحنات
- نظام التتبع والمراقبة

---

### 8. وكيل السلامة والأمن
**المهام:**
- تقييم مخاطر السلامة
- وضع إجراءات السلامة
- التحقق من متطلبات الأمن
- إدارة التأمين

**المخرجات:**
- تقرير تقييم المخاطر
- خطة السلامة التفصيلية
- إجراءات الطوارئ
- بوالص التأمين

## 🛠️ قوائم الإجراءات والمعدات التفصيلية

### أ) مشروع التوريد - الإجراءات خطوة بخطوة

#### المرحلة 1: ما قبل التوريد (Pre-Procurement)
```
□ 1.1 تحديد الاحتياج
   - نوع المعدن المطلوب
   - الكمية المطلوبة
   - المواصفات الفنية
   - الجودة المطلوبة

□ 1.2 دراسة السوق
   - البحث عن الموردين
   - مقارنة الأسعار
   - تقييم الموردين
   - طلب عينات (إن أمكن)

□ 1.3 إعداد الميزانية
   - حساب التكلفة الإجمالية
   - الموافقة المالية
   - فتح اعتماد مالي (إن لزم)

□ 1.4 التحقق من الامتثال
   - مراجعة المتطلبات القانونية
   - التحقق من التراخيص
   - التأكد من الامتثال الشرعي
```

#### المرحلة 2: التعاقد والشراء
```
□ 2.1 إعداد العقد
   - تحديد الشروط والأحكام
   - شروط الدفع
   - مدة التوريد
   - شروط الجودة والمطابقة

□ 2.2 المفاوضات
   - التفاوض على السعر
   - التفاوض على شروط التسليم (Incoterms)
   - الاتفاق على طريقة الدفع

□ 2.3 التوقيع والاعتماد
   - توقيع العقد من الطرفين
   - الختم والتوثيق
   - إصدار أمر الشراء (Purchase Order)

□ 2.4 الدفع
   - دفع مقدم (إن وجد)
   - فتح اعتماد مستندي (L/C) إن لزم
   - توثيق عملية الدفع
```

#### المرحلة 3: الشحن والاستلام
```
□ 3.1 ترتيب الشحن
   - اختيار شركة الشحن
   - حجز مساحة الشحن
   - تجهيز الوثائق

□ 3.2 التتبع والمراقبة
   - تتبع الشحنة عبر GPS/RFID
   - التواصل مع شركة الشحن
   - استقبال التحديثات

□ 3.3 الإجراءات الجمركية
   - تقديم المستندات للجمارك
   - دفع الرسوم الجمركية
   - الحصول على إفراج جمركي

□ 3.4 الاستلام والفحص
   - استلام البضاعة في الموقع
   - فحص الكمية
   - فحص الجودة
   - توثيق الحالة (صور/فيديو)
   - قبول أو رفض البضاعة

□ 3.5 التخزين
   - نقل البضاعة للمستودع
   - تسجيل في نظام المخزون
   - وضع بطاقات تعريف
   - ربط أجهزة IoT للمراقبة
```

#### المرحلة 4: إدارة الجودة
```
□ 4.1 فحص الجودة الأولي
   - فحص عينة عشوائية
   - قياس الوزن الفعلي
   - فحص المطابقة للمواصفات

□ 4.2 الفحص المعملي (إن لزم)
   - فحص التركيب الكيميائي
   - فحص الشوائب
   - الحصول على شهادة الفحص

□ 4.3 إصدار شهادة الجودة
   - إعداد تقرير الفحص
   - اعتماد شهادة الجودة الداخلية
   - الحصول على شهادة SASO (إن لزم)

□ 4.4 معالجة عدم المطابقة
   - توثيق أي عدم مطابقة
   - التواصل مع المورد
   - إجراءات الإرجاع أو التعويض
```

#### المرحلة 5: التسويق والبيع
```
□ 5.1 النشر على منصة شيخة
   - تصوير المنتج
   - كتابة الوصف الدقيق
   - تحديد السعر
   - نشر الإعلان

□ 5.2 التواصل مع المشترين
   - الرد على الاستفسارات بصدق
   - تقديم العينات (إن لزم)
   - المفاوضات

□ 5.3 إتمام البيع
   - إعداد عقد البيع
   - استلام الدفع
   - تجهيز البضاعة للتسليم

□ 5.4 التسليم
   - تعبئة آمنة
   - إصدار مستندات التسليم
   - التسليم للعميل أو شركة الشحن
   - تأكيد الاستلام
```

## 🚛 آلية التوصيل الآمن بصدق وأمانة

### 1. مبادئ التوصيل الإسلامي

#### الصدق
```
✓ وصف دقيق للمنتج (الوزن، الجودة، الحالة)
✓ الإفصاح عن أي عيوب أو نقص
✓ عدم المبالغة في الوصف
✓ الوضوح في شروط التسليم
✓ الالتزام بالمواعيد المحددة
```

#### الأمانة
```
✓ الحفاظ على البضاعة من التلف
✓ عدم التلاعب بالكميات
✓ التعامل بحرص مع ممتلكات الغير
✓ حفظ خصوصية معلومات العميل
✓ إعادة أي زيادة للعميل
```

#### العدل
```
✓ عدم التمييز بين العملاء
✓ الأسعار العادلة
✓ معاملة منصفة للجميع
✓ حل النزاعات بالعدل
```

---

### 2. إجراءات التوصيل الآمن خطوة بخطوة

#### أ) مرحلة ما قبل التوصيل
```
□ تأكيد الطلب مع العميل
   - تفاصيل المنتج
   - الكمية المطلوبة
   - موعد التسليم
   - مكان التسليم

□ فحص البضاعة قبل الشحن
   - التأكد من المطابقة
   - فحص التغليف
   - قياس الوزن النهائي
   - توثيق بالصور (timestamp)

□ تجهيز الوثائق
   - فاتورة البيع
   - قائمة التعبئة
   - شهادة الجودة (إن وجدت)
   - مستند التسليم

□ اختيار وسيلة النقل المناسبة
   - حسب الحجم والوزن
   - حسب المسافة
   - حسب طبيعة المنتج
```

#### ب) مرحلة التغليف الآمن
```
□ التغليف الداخلي
   - استخدام مواد حماية مناسبة
   - فصل القطع عن بعضها (إن لزم)
   - وضع مواد عازلة

□ التغليف الخارجي
   - صناديق قوية ومحكمة
   - ربط محكم بالأشرطة
   - أختام أمان (Security Seals)

□ وضع العلامات
   - اسم المرسل والمستقبل
   - رقم الطلب
   - تحذيرات (هش، ثقيل، اتجاه الأعلى)
   - باركود/QR Code
```

#### ج) مرحلة الشحن والنقل
```
□ تسليم للناقل
   - التأكد من هوية السائق
   - توثيق التسليم (صور + توقيع)
   - تفعيل جهاز GPS
   - مشاركة رابط التتبع مع العميل

□ المتابعة اللحظية
   - تتبع الموقع عبر GPS
   - التواصل مع السائق
   - تحديثات للعميل

□ الالتزام بمعايير السلامة
   - القيادة الآمنة
   - الالتزام بأوزان الحمولة
   - التوقف في أماكن آمنة
   - عدم ترك المركبة بدون حراسة
```

#### د) مرحلة التسليم النهائي
```
□ التواصل قبل الوصول
   - إخطار العميل قبل 30 دقيقة
   - تأكيد التواجد في موقع التسليم

□ عند الوصول
   - التحقق من هوية المستلم
   - فك التغليف أمام العميل (إن طلب)
   - وزن البضاعة أمام العميل

□ التسليم والتوثيق
   - تسليم البضاعة والوثائق
   - الحصول على توقيع الاستلام
   - تصوير عملية التسليم
   - تسجيل أي ملاحظات

□ التأكيد
   - تأكيد التسليم في النظام
   - إرسال إشعار للعميل
   - update حالة الطلب على المنصة
```

#### هـ) ما بعد التسليم
```
□ المتابعة
   - الاتصال بالعميل بعد يوم
   - التأكد من الرضا
   - حل أي مشاكل

□ التوثيق النهائي
   - حفظ جميع المستندات
   - أرشفة الصور والفيديو
   - تحديث السجلات

□ التحسين المستمر
   - جمع ملاحظات العميل
   - تحليل أي مشاكل
   - تطبيق التحسينات
```
