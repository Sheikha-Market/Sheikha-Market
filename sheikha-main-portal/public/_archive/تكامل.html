التكامل — Integration

ربط سوق شيخة مع أنظمتك ومنصاتك: الذكاء الصناعي، جوجل، أمازون، أوراكل، ساب، و ERP. نقطة واحدة لسهولة التسويق والوصول.

لماذا التكامل؟ شركة واحدة، لوحة تحكم واحدة، وكل خيارات الربط من مكان واحد — للمصاهر والشركات والمطورين. اختر المنصة أو النظام الذي تستخدمه واربطه مع سوق شيخة.

🤖 منصات الذكاء الصناعي (AI)

ربط سوق شيخة مع الذكاء الصناعي — أسعار وعروض وطلبات من واجهة واحدة.

  • مساعد شيخة: محادثة ذكية واقتراح أسعار مدمج في الموقع.
  • Sheikha Partner API: واجهة REST/JSON لربط أي نظام أو روبوت — عروض، طلبات، تقارير.
  • Webhooks (لاحقاً): إشعار فوري عند طلب جديد أو تغيير حالة — لأتمتة الرد.
  • تكامل مع ChatGPT / Claude (مستقبلاً): إمكانية Action أو Plugin لاستعلام أسعار وعروض شيخة من منصات الذكاء.
مساعد شيخة الذكي واجهات API

🔍 جوجل (Google)

شيخة تعمل مع بيئة جوجل — بحث، جداول، سحابة.

  • Google Search / SEO: الموقع مُحسّن للبحث؛ ظهور أغنى في نتائج جوجل.
  • Google Workspace / Sheets (لاحقاً): تصدير عروضك أو تقاريرك إلى Google Sheets.
  • Google Cloud: استدعاء Sheikha API من سكربتات أو خدمات على Google Cloud.
الموقع — يظهر في جوجل

📦 أمازون (Amazon)

تكامل مع أنظمة أمازون — API واحد لسوق السكراب والمعادن.

  • منطق SP-API / ربط البيانات: ربط عروض وطلبات شيخة مع أنظمة تستخدم أمازون.
  • Amazon Web Services (AWS): Sheikha API يعمل مع AWS — استدعاء من Lambda أو EC2.
Sheikha Partner API

☁️ أوراكل (Oracle)

ربط Oracle ERP و Oracle Cloud مع سوق شيخة — عروض وطلبات سكراب في نظامك.

  • Oracle ERP / Oracle Cloud: استيراد احتياجات الشراء، تصدير أوامر الشراء، مزامنة العروض والطلبات عبر Sheikha API.
  • Oracle Integration Cloud: استخدام Sheikha API ضمن سير عمل أوراكل.

🏢 ساب (SAP)

ربط SAP مع سوق شيخة — مشتريات سكراب ومعادن من داخل SAP.

  • SAP ERP / S/4HANA: ربط وحدة المشتريات أو المواد الخام مع شيخة — استيراد عروض، تصدير أوامر شراء، تقارير.
  • SAP Business Technology Platform: تكامل Sheikha API مع سير عمل SAP.
دليل التكامل مع ERP

📊 أنظمة ERP عامة

أي ERP — شيخة تتكامل مع نظامك عبر API أو ملفات. Odoo، Microsoft Dynamics، أو أي نظام يدعم REST API أو استيراد/تصدير.

  • Sheikha Partner API (REST/JSON): عروض، طلبات، تقارير — ربط من أي نظام.
  • استيراد/تصدير ملفات (CSV/Excel): رفع عروض؛ تحميل تقارير من لوحة التحكم دون برمجة.
  • مفتاح API (API Key): للشركات والمطورين — طلب مفتاح للوصول إلى API.
