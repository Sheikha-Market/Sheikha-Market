ابدأ التكامل في دقائق

JavaScript
Python

Go
// تثبيت Sheikha Core SDK
// npm install @sheikha/core @sheikha/market @sheikha/contracts

import { SheikhaClient } from '@sheikha/core';
import { MarketSDK } from '@sheikha/market';
import { ContractSDK } from '@sheikha/contracts';

// تهيئة العميل
const client = new SheikhaClient({
  apiKey: 'sk_live_your_api_key',
  environment: 'production',
  locale: 'ar'
});

// جلب أسعار المعادن اللحظية
const market = new MarketSDK(client);
const prices = await market.getLivePrices({
  metals: ['gold', 'silver', 'copper'],
  currency: 'SAR'
});

console.log(prices);
// → { gold: { price: 285.40, change: +1.2%, exchange: 'LBMA' }, ... }

// إنشاء عقد رقمي
const contracts = new ContractSDK(client);
const contract = await contracts.create({
  type: 'supply',
  parties: { buyer: 'buyer_id', seller: 'seller_id' },
  items: [{ metal: 'copper', qty: 5000, unit: 'kg' }],
  terms: { payment: 'net_30', delivery: 'CIF' }
});

console.log(contract.id); // → 'cnt_2026_abc123'

المعاملات المطلوبة في كل طلب API

المعامل النوع إلزامي الوصف
Authorization header ✓ نعم Bearer <API_KEY> — مفتاح API الخاص بك
X-Sheikha-Version header اختياري إصدار API المطلوب (الافتراضي: الأحدث)
X-Request-ID header اختياري معرف الطلب للتتبع (UUID)
locale query اختياري لغة الاستجابة: ar، en (الافتراضي: ar)
