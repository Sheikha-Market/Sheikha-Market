```bash

```

### **المشكلة: Copilot لا يعمل**
```bash
# تحقق من الدخول
Ctrl+Shift+P
ابحث عن: GitHub: Sign in
سجل الدخول بحسابك GitHub
```

### **المشكلة: Debugging لا يعمل**
```bash
# الحل
عادة المشكلة في .env
تأكد من وجود:
NODE_ENV=development
DEBUG=sheikha:*
```

### **المشكلة: Performance بطيء**
```bash
# الحل

2. قلل عدد Extensions المفعّلة
3. أعد تشغيل VS Code: Ctrl+Shift+P > Reload Window
```
