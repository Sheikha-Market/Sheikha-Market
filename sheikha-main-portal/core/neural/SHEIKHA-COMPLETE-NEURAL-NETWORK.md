```

├── FIN  المالية والمحاسبة     ← الزلزلة:7-8 (مثقال ذرة)
├── INV  المخزون والمستودعات   ← يوسف:47 (تزرعون سبع سنين)

├── PRD  تخطيط الإنتاج         ← الملك:1 (بيده الملك)
└── RPT  التقارير والتحليلات   ← الحديد:4 (بما تعملون بصير)
```
