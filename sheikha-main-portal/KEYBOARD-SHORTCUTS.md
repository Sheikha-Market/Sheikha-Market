## 🐛 **Debugging** (أدوات البحث عن الأخطاء)

| اختصار | الفائدة |
|--------|--------|
| `F5` | **تشغيل Debug** 🎯 |
| `F9` | إضافة/حذف Breakpoint |
| `F10` | خطوة واحدة (Step Over) |
| `F11` | الدخول للدالة (Step Into) |
| `Shift+F11` | الخروج من الدالة (Step Out) |
| `Ctrl+F5` | تشغيل بدون Debug |
| `F6` | Pause |
| `Ctrl+Shift+F5` | Restart Debugger |

**استخدام الـ Debug:**
```bash
1. اضغط F5
2. اختر أحد الـ 6 Debug Modes
3. اضغط F9 على السطر المريب
4. البرنامج سيتوقف هناك
5. استخدم F10 للخطوات
6. اسأل عن قيمة متغير بـ Hover
```
