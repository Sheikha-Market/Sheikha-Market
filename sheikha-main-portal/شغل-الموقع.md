# تشغيل الموقع — سوق شيخه

## الطريقة 1: من الطرفية (Terminal)

1. افتح **Terminal** (طرفية Mac) أو **Terminal.app**.
2. نفّذ:

```bash
cd "/Users/salmanalrajeh/Library/CloudStorage/OneDrive-Personal/Sheikha/sheikha-main-portal"
npm start
```

أو من مجلد شيخه الرئيسي:

```bash
cd "/Users/salmanalrajeh/Library/CloudStorage/OneDrive-Personal/Sheikha"
./تشغيل-الشبكة-والموقع.sh
```

3. انتظر حتى تظهر رسالة أن السيرفر يعمل.
4. افتح المتصفح على: **http://localhost:8080**

---

## الطريقة 2: إذا ظهر "npm: command not found"

ثبّت Node.js أولاً (مرة واحدة):

- من [nodejs.org](https://nodejs.org) — حمّل النسخة LTS وثبّتها.
- أو عبر Homebrew: `brew install node`
- أو عبر nvm: `curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash` ثم `nvm install --lts`

ثم أعد تشغيل الطرفية ونفّذ الأمر أعلاه.
