# ترخيص شيخة — جميع الحقوق محفوظة

حقوق الملكية الفكرية لهذا المشروع محفوظة للمالك:
**سلمان أحمد بن سلمان الراجح**.

## شروط الاستخدام
1. يمنع النسخ أو إعادة التوزيع أو إعادة البيع بدون إذن كتابي صريح.
2. يمنع إزالة إشعارات الملكية أو تعديلها.
3. يسمح بالاستخدام التشغيلي المرخص فقط ضمن حدود الاتفاق.
4. أي استخدام غير مصرح يعد تعديًا على الحقوق.

## إخلاء المسؤولية
يُقدَّم النظام كما هو مع التزام المالك بالتطوير والتحسين وفق الضوابط الشرعية والقانونية.

## المرجعية
- الكتاب والسنة.
- لا ضرر ولا ضرار.
