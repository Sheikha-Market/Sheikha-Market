## 4. تدفق المعلومات بين اللوحات

### 4.1 رحلة المستخدم
```
زائر → تسجيل دخول → [لوحة مستخدم] → يتصفح السوق → يطلب → يتتبع → يقيّم
     ↓
  تسجيل شركة → [لوحة شركة] → يعرض منتجات → يستقبل طلبات → يشحن
     ↓
  تسجيل حكومي → [صفحة حكومية] → مشاريع/أسعار/مجتمع/منتجات
```

### 4.2 تدفق البيانات بين الأنظمة
```
[سوق شيخة :8080] ←→ [سوق شامل :23000]
       ↕                    ↕
[لوحة مستخدم]         [أسواق خارجية]
       ↕
[لوحة شركة]
       ↕
[لوحة أدمن]
       ↕
[صفحات إدارية _admin/]
```

### 4.3 دورة المعاملة التجارية
```
عرض منتج → طلب شراء → عرض سعر → موافقة → فاتورة → دفع → شحن → تسليم → تقييم
    ↓         ↓          ↓         ↓        ↓      ↓       ↓       ↓       ↓
 products  orders    quotes   contracts  invoices  pay   logistics  track  review
```

                    ┌─────────────────┐
                    │   المالك/المدير  │

              │             │             │
    ┌─────────┼─────────────┼─────────────┼──────────┐
    │         │             │             │          │
┌───▼──┐ ┌───▼──┐  ┌───────▼─────┐ ┌────▼───┐ ┌───▼────────┐
│إدارة │ │إدارة │  │إدارة سلاسل │ │إدارة   │ │إدارة       │
│السوق │ │CRM & │  │الإمداد      │ │الأمان  │ │الجودة      │

### الإدارات الثمانية:
1. **إدارة السوق** — المنتجات، التسعير، الأسواق
2. **إدارة CRM والتسويق** — العملاء، الحملات، المحتوى
3. **إدارة سلاسل الإمداد واللوجستيات** — الشحن، المستودعات، التتبع
4. **الإدارة المالية** — المدفوعات، الزكاة، الفواتير، ERP
5. **الإدارة التقنية** — البرمجة، API، AI، DevOps
6. **إدارة الأمان والحماية** — الأمان السيبراني، النسخ الاحتياطي
7. **إدارة الجودة والامتثال** — المعايير، السياسات، التراخيص
