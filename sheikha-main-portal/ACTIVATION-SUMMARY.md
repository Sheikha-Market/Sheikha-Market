### 🐛 **5. Advanced Debugging (أدوات Debugging متقدمة)**

#### 6 Debugging Configurations
1. **🚀 Sheikha: Launch Server (NVIDIA-first)**
   - مع دعم NVIDIA CUDA
   - أمثل للتطوير المتقدم

2. **🐛 Debug: Server with Profiling**
   - مع جمع بيانات الأداء
   - لقياس استهلاك المعالج

3. **🔍 Debug: With Memory Profiling**
   - تحليل استهلاك الذاكرة
   - اكتشاف memory leaks

4. **⚡ Development: Watch Mode**
   - إعادة تحميل تلقائية
   - للتطوير السريع

5. **🔗 Attach: to Running Process**
   - الالتصاق بعملية موجودة
   - للـ Remote Debugging

6. **🎯 Full Stack Debug**
   - دمج جميع أدوات الـ Debug
   - للتطبيقات المعقدة

#### Debugging Features
- ✅ **Breakpoints** - نقاط توقف ذكية
- ✅ **Watch Variables** - مراقبة المتغيرات
- ✅ **Call Stack** - تتبع استدعاءات الدوال
- ✅ **Memory Inspection** - فحص الذاكرة
- ✅ **Performance Profiling** - قياس الأداء

**الاستخدام:**
```bash
F5 -> تشغيل Debug
F9 -> إضافة/حذف Breakpoint
F10 -> خطوة واحدة
F11 -> الدخول للدالة
Shift+F11 -> الخروج من الدالة
Ctrl+Shift+D -> فتح Debug Panel
```

## 🔐 **الأمان والخصوصية**

✅ **بيانات آمنة محلياً:**
- الكود يُعالج محلياً أولاً
- Copilot يستخدم HTTPS آمن
- لا ترسيل ".env" أو ملفات حساسة

✅ **تحكم كامل:**
```json
"github.copilot.enable": {
    "*": true,           // مفعل في كل الملفات
    ".env": false,       // ❌ معطل في .env
    "*.secret": false    // ❌ معطل في الملفات السرية
}
```
