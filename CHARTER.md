# ميثاق منظومة شيخة (Sheikha Charter)

## 1) المرجعية
هذا المشروع محكوم بـ:
1. كتاب الله
2. سنة النبي ﷺ
3. الضوابط النظامية المعمول بها
4. الجودة التقنية والانضباط التشغيلي

## 2) مواد إلزامية
1. يمنع أي تدفق أو ميزة تتضمن ربا صريح أو ضمني.
2. يمنع الغرر والجهالة في العقود والتسعير.
3. يمنع الغش أو الإخفاء أو التضليل.
4. يمنع الاحتكار والإضرار بالمنافسة العادلة.
5. تلزم الشفافية في الرسوم والعمولات والالتزامات.
6. لا يُطلق أي تغيير قبل فحص أمني وتشغيلي وشرعي.

## 3) ميثاق هندسي
- كل تغيير يجب أن يكون:
  - موثقًا
  - قابلًا للتتبع
  - قابلًا للاختبار
- لا أسرار داخل الكود (Secrets in env only).
- أي endpoint حساس يحتاج تحقق صلاحيات واضح.

## 4) ميثاق البيانات
- تقليل البيانات (Data Minimization).
- عدم جمع بيانات بلا غرض تشغيلي واضح.
- حماية الخصوصية والتشفير أثناء النقل والتخزين.

## 5) ميثاق الإطلاق
- صحة الخدمة (Health) = OK
- عدم وجود أخطاء حرجة
- جاهزية المراقبة والتنبيه
- خطة تراجع (Rollback) جاهزة

## 6) دورة المراجعة
- مراجعة تقنية أسبوعية
- مراجعة مؤشرات تشغيل يومية
- مراجعة امتثال شرعي شهرية
- توثيق قرارات الحوكمة في سجل ثابت
